# 2025-08-13 选牛股的核心逻辑

> 老师说这是八月份最重要的一课。

## 异动（移仓异动）

主力资金开始介入某只股票的早期信号。通过量价配合识别。

- **特征**：出现异常的成交量变化（堆量、放量），价格开始有节奏地运动
- **初期的异动**：最好的介入时机，主力刚入场，还没拉升
- **堆量异动**：连续多根阳线带着温和放量，一坨一坨地堆上去
- **原文金句：**"初期的异动是最好的"

## 堆量（堆量建仓）

主力建仓期的成交量特征：多根阳线带着温和放量，形成一坨一坨的成交量堆。

- **对比**：不是单根巨量，而是连续多根小到中阳线堆出来的量
- **三连跳建仓**：三根连续的堆量阳线形成建仓信号
- **堆量之后等B1**

## 一波流

主力一气呵成拉升，中间不给你上车机会。

- **特征**：BBI线上一直涨，不破BBI线，没有显著回调
- **处理**：如果你第一波没进去，就算了，不追
- **建仓和拉升合在一起**：一边买一边拉，建仓和拉升不分开
- **原文金句：**"一波流的票不让你外盘散户进场的，所以不要研究它"

## 节奏仓位管理

根据每日钩数（B1数量/金叉数量）决定仓位大小。

| 每日钩数 | 操作 | 仓位 |
|:---------|:-----|:-----|
| 2000+ | 可以买4个票 | 重仓 |
| 1000+ | 买1-2个 | 中等 |
| 700 | 只买1个 | 轻仓 |
| 300 | 观察，只选完美的 | 极轻或空仓 |

- **核心**：钩多就多做，钩少就少做，做到后面完美度的标准越来越高
- **原文金句：**"节奏很重要"

## 完美图形评分体系

用NBA球员类比股票图形的"身体素质"，从弱到强：

| 级别 | 类比 | 特征 |
|:----|:-----|:-----|
| 徐杰 | 单薄 | 身材小、量能薄、容易熄火 |
| 胡明轩 | 合格 | 有一定厚度，勉强可以 |
| 拉希德华莱士 | 70分 | 还行，但不够顶级 |
| 本华莱士 | 75-80 | 二次发育后变强，有上限 |
| 莫兰特 | 小号乔丹 | 飞在天上像乔丹但整体小一号 |
| 杜兰特 | 90+ | 优雅、标准、教科书 |
| 詹姆斯 | 99 | 顶级，完美图形 |
| 蔡恩 | 不讲道理 | 强壮但暴力，不跟你讲逻辑 |

- **核心**：见了詹姆斯，就知道徐杰不行了——"就怕货比货"
- 同一时间同样条件下选股，选未来3天到1周表现最好的

## 完美图形的特征

- 呼吸同呼吸共命运（N型+量能配合）
- 红肥绿瘦：阳量>阴量
- 缩量时缩到极致，放量时温和放大
- 没有"价格涨量跌"的背离
- 没有大绿棒子（放量巨阴）
- 建仓波不放涨停

## 不完美的图形特征

- **价格在涨，量在下跌** = 不完美，选出来也不干
- 单薄：量能薄弱的图
- 阴阳不调：阳线少阴线多
- 阶梯量大且时间长：需要足够时间消化
- 层峦叠嶂：上面大量套牢盘

## 异动与呼吸（量价配合）

一个标准的异动-建仓-拉升流程：

```
异动（初期的信号，来人了）
  → 堆量（建仓期，连续小阳堆量）
    → 缩量（洗盘，呼吸中的"呼"）
      → B1（买入点）
        → 拉升（放量上攻，量随价升）
          → 顶部放量（出货）
```

- **堆量加速建仓**：建仓后期，放量加速买入
- **缩量B1**：堆量后缩量下来的B1是最佳买点
- **放量再缩量**：健康的循环

## 强势板块轮动节奏

大盘调整前后板块轮动的规律：

- **市场热时**：算力→芯片→光刻胶/光刻机→AI软件
- **大盘调整时**：医药、化工、小金属（防御型）
- **防御型的逻辑**：涨价＝确定性。涨价必然带来业绩确定性，确定性就能防守
- **反内卷的本质是涨价**：CPI要转正，所以要涨价
- **医药后面还有好多**：防御时医药接过来

## B1评分系统（量化思路）

通过建立打分系统筛选B1，标准越高越好。

- 完美图形=高分（80+），单薄图形=低分（40-50）
- 同一批B1中只选最高分的1-2个
- 量化系统打分：结合量、价、形态、N型、筹码等多维度
- 可以用通达信/同花顺结合AI工具辅助

## 阶梯量之后的SB1

大的阶梯量出现后，不能马上做B1，需要足够长的时间消化。

- **标准阶梯量**：连续多根阴线逐级缩量，形状像楼梯
- 阶梯量后等SB1（超级B1）——需要足够时间
- 时间不够的阶梯量不做

## 势能降低

股票上涨过程中，上涨力度在衰减。

- **特征**：K线开始松散，底下的获利筹码开始出货
- **识别**：连续B1往下走很多，说明势能在降低
- **处理**：注意防卖飞，该减仓减仓

## 一波流建仓拉升合一的票

主力边买边拉，建仓和拉升放在一起。

- **特征**：BBI线上面一直买，一边买一边拉
- **识别**：在拉升前可能真的给一个B1
- **案例**：从63到153，一回调打B1

## 大队长思维（腾讯类比）

确定性强的票如腾讯，所有概念都跟着它结账。

- 游戏、微信、算力、芯片，哪波来了都能跟上
- 所有A股徒子徒孙闹到最后，统一到队长这结账
- 大资金来了没得选，只能配队长这类确定性标的

## 本课系统贡献

**底层认知层贡献：**
- 防御板块的底层逻辑是**涨价**（确定性）而非位置低
- 反内卷的本质是涨价，CPI转正需要涨价
- 大队长思维：确定性强的龙头票是所有资金最终的归宿

**择时层贡献：**
- **节奏仓位管理**：按每日钩数（B1/金叉数量）动态调整仓位，钩多就多做，钩少标准严
- 大盘调整前后板块轮动节奏：科技→光刻→软件→医药防御
- B1从2000逐渐降到300的过程，节奏从重仓→半仓→轻仓→观察

**买入层贡献：**
- **完美图形评分体系**：用NBA球员类比、凭"身体素质"评分选股
- **异动识别体系**：初期的异动→堆量→缩量B1→拉升的完整流程
- 不完美图形的排除标准：量价背离、单薄、阴阳不调、阶梯量
- B1评分系统思路：量化打分，80分以上才干

**分析依据贡献：**
- **势能降低识别**：K线松散、获利筹码出货的信号
- 堆量建仓 vs 一波流建仓拉合一的区分
- 价格涨量下跌=不完美的核心原则
- 阶梯量之后的SB1：需要足够时间消化
- 异动后的B1 vs 无异动的B1质量差异
- 两个稳定币走势对比：宁波（阳量足、阴量缩）vs 另一个（阴量多）

**认知与纪律贡献：**
- "降低预期，才不入急门"——一个月一根就够了
- "先降低预期，存银行一年才两个点"——不要对收益率有不切实际的幻想
- "牛市还没来"——现在是热身阶段
- "不要在我的专业领域挑战我"
- 这个世界是动态的，一定要动态地看问题
