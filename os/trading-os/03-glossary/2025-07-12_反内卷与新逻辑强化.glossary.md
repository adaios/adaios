# 交易术语库：2025-07-12 反内卷与新逻辑强化

## B1（买点1）——增量更新

**增量更新（2025-07-12 反内卷与新逻辑强化）：**

### 少妇战法B1的精髓定义
- **定义**：B1位置买在短期获利盘或卖盘枯竭的地方。短期向下杀跌动能已释放，该卖的人已卖完。极少出现B1买进就跌停（除非实际控制人被立案等黑天鹅，概率极小）。
- **解决散户第一个问题——一买就被套**：B1解决了散户以前没有交易策略时买完当天被套3-4个点、第二天波动到-7%就慌了的问题。B1勾大幅值缩量下来，短期最快速向下杀跌的动能已经释放。
- **解决散户第二个问题——盈亏比**：向上10%，向下2%，盈亏比5:1。向上20%，向下2%，战损比10:1。
- **市场意义**：少妇战法最接近市场本质，最适合散户的正路策略。

### 超级B1（超级买点）——新增概念
- **定义**：B1位置的升级形态。缩量横住 -> 放量下跌（短期分歧）-> 又横住 -> 缩量 -> 此时K线勾大幅值，即为超级B1。
- **特征**：
  - 图形非常难看，一般破前低或破小平台，出现破位放量阴线
  - 大多数人在此位置被吓走
  - 等出现小十字星（反转K/止损K），底下缩量勾仍大幅值，即为超级买点
- **操作方式**：超级B1买完后在进场K线最低点下方设止损。不是稳妥的选择，是要赌的。超级B1之后往往直线拉升，进入主升浪。
- **案例流程**：破位放量阴线 -> 小十字星缩量勾大幅值 -> 进场 -> 放巨量把最后跟风盘打走 -> 直接一字板拉升。这种走势是标准的麒麟会玩法——逼真到所有人都会相信这是恶庄。
- **市场意义**：很多牛股在启动最后一波大浪之前靠超级B1启动。超级B1出现一般将进入麒麟会真正的主升浪。
- **注意**：超级B1不是发财就是发疯，必须设止损。

### N型结构中的量价关系与B1（深化）
- **上涨方向的N型结构**：找极致缩量点，不破前低则为极好入场点（B1）。
- **下降方向的N型结构**：找放量点（卖的人多但买的人也多了，容易形成短期底部）。
- **超级B1在N型中的演变**：缩量横住 -> 放量下跌（短期分歧）-> 缩量再横住（分歧后一致）-> 回到B1 -> 超级B1。

## N型结构的本质——新术语

- **定义**：N型结构的本质是不断抬高市场对手盘的跟风成本。
- **运作逻辑**：麒麟会或主导资金在拉升一个票时，每到一个阶段完成市场流通筹码交换的过程——让底部低位的跟风筹码换到当前价格位置的更高成本。通过呼吸（上涨再回调）不断换筹码，让跟风盘成本越来越高。
- **具体过程**：
  1. 从10块拉到30时，把底部10块成本的人换到30块
  2. 跟风盘成本变成30块后，不会轻易走
  3. 再拉5%，当天跌4%，跟风盘中一半被洗走换35块成本
  4. 直接拉涨停——新进跟风盘成本35块，获利不大不会卖
- **市场意义**：主力不能让底部低成本的人一直在车上。如果底部获利盘不换手，拉到50谁出货？主力花了钱却给底部10块的人抬轿，这不成立。
- **反面案例**：米儿洗不走，因为成本已降到负6负7。主力必须洗掉36-59之间的跟风盘，通过洗盘将跟风盘成本逐步抬高。
- **对散户的意义**：理解了这一本质，就不会总问这个还能拿吗那个还能拿吗。

## 机构入场（价值发现）vs 游资（短线投机）——增量更新

**增量更新（2025-07-12 反内卷与新逻辑强化）：**

### 机构入场逻辑深化
- **特征**：机构资金入场时不太在意跟风盘或散户，只求快速配置进去。一回调机构觉得稳了就会接着配置。
- **不在意游资**：即使游资跟进拉升，机构第二天会松手让游资离场，等两天再接着买。
- **为什么**：机构干的是价值发现，不是短线投机。

### 游资本质
- **言论判断**：外面游资言论可以听但不能信。他们灌输游资玩法是为了培养韭菜接盘2000-5000万资金。
- **追涨停板本质**：中高位涨停板都是出货。低位涨停板无所谓看不出来。天上的涨停板都是出货。
- **换手率的真相**：高位放量换手率卖的是低位潜伏的机构、麒麟会、游资或其它散户的筹码。如果后面还能涨意味着来了新人击鼓传花，否则都叫出货。
- **游资与主播类比**：把认识的游资名字当成抖音主播或B站UP主就对了。游资席位有时专门用来出货（折价5-10%的大宗交易）。

### 量化与游资的关系
- **量化干的就是游资**：量化针对游资的一招鲜做出反制措施实现套利。
- **骂量化最多的是游资自媒体**：大机构很少骂量化。散户认为不合理的事情长期存在必有其原因——存在即合理。
- **游资是量化的大对手盘**：散户已经被套没钱了，游资这块肉被量化盯上了。

## 成功才是成功的妈妈——新术语

- **定义**：否定失败是成功之母的传统说法，强调正向反馈和成功经验的积累才是真正带来成功的因素。
- **市场意义**：散户在交易中不要沉迷于从失败中学习，而要关注正确的操作带来的正向效果。
- **案例**：龙头战法的正确理解——微信、华纳、娜娜这种机构票的稳健上涨才是真正的龙头战法，而不是游资追涨停板。

## 跟风盘成本交换理论——新术语

- **定义**：主力通过N型结构的呼吸（拉升-回调-再拉升），不断让低位跟风盘的筹码换到更高的成本区间，为后续拉升和出货创造条件。
- **特征**：
  - 每一波拉升后必须让底部跟风盘换手
  - 跟风盘成本越高，拉升越顺畅
  - 跟风盘成本接近当前价格时，小幅波动洗不走
- **市场意义**：解释了为什么主力要洗盘、为什么有些票洗得特别狠、为什么有些票能一路拉升不回调。
- **与普通洗盘的区别**：洗盘是短期行为，成本交换是贯穿整个N型结构的长周期动作。

## 缩量的真假判断（深化版）——增量更新

**增量更新（2025-07-12 反内卷与新逻辑强化）：**

### 缩量的核心判断
- **缩量做不了假**：放量可以用左账户卖右账户买（左兜换右兜）做假，但缩量骗不了人。
- **缩量的两个性质（按位置区分）**：
  1. N型结构单边下跌时缩量不要碰——买盘非常少，稍微有人卖就下跌。事物沿阻力最小方向前进，缩量下跌意味着阻力向下。
  2. N型结构上涨时任何缩量都意味着短期卖盘枯竭，稍微有人买一点就会继续上涨。

### 放量判断（对比补充）
- 连续N型结构下降时找放量点（卖的人多但买的人也多了，容易形成短期底部）。
- N型结构顶部放量下跌一律是真出货。连续上涨后巨量长阴线或跌停板，不要判断是不是洗盘，就是出货。如果第二天反弹说明没出完，还会再拉一波再出。

### 呼吸模式与量价
- 正常呼吸节奏：放量上涨 -> 缩量下跌 -> 放量上涨 -> 缩量下跌。
- 缩量横住后突然放量下跌：除非是放量跌停，否则一般是短期分歧，分歧后又是一致，后面找缩量点，如果又是B1就是很好的右侧买点。

## 锚定效应（交易心理）——新术语

- **定义**：散户在市场中容易因为某些极端案例（如华银电力、恒宝、浙江建投等游资票），形成追涨停也能赚钱的锚定认知，忽略了5000分之1的成功概率。
- **市场意义**：游资利用锚定效应教育市场和培养韭菜。散户应该认识到极端案例不是普遍规律。
- **常见误区**：看到一两个游资票连续涨停就认为追涨停板是可行策略。

## 市场跟随原则——新术语

- **定义**：做股票的本质是跟随不是预判。
- **具体内容**：跟随市场，不要预判市场怎么走。按照自己的交易纪律执行。票顺着预期往上走了拿着别说话。如果不符预期拍掉换新的。节奏不能乱，不要因为走势不符合预期就生气。
- **市场意义**：解决了散户被盘面带情绪的问题。
- **类比**：谁是县长不重要，重要的是当县长夫人。票是谁给的10%不重要，重要的是拿到10%。

## 筹码纯度 / PURE筹码——新术语

- **定义**：当麒麟会拿了70%的流通筹码、外面还有30%的筹码时，需要通过洗盘让这30%筹码的成本控制在一个非常精确的区间内，使整个盘子变得pure（干净）。
- **特征**：筹码越pure，拉升越顺畅。如果各个价位都有跟风盘，拉升非常困难。
- **市场意义**：解释了主力为什么在最后主升浪启动前要把跟风盘洗到亲妈都不认识——为了减少拉升阻力。

## 高位获利盘出货逻辑——新术语

- **定义**：在股价大幅上涨后（如从底部54涨到1108），底部获利盘的卖出动力极强。高位票每拉一根阳线，底部获利的大资金就大量出货。
- **特征**：
  - 高位票必须敏感：拉几根阳线就要考虑减仓。
  - 越涨越卖，不要在高位拿着不动做交易性机会。
  - 大资金必须分批出货，一笔单子出去可能直接砸跌停。
- **市场意义**：区分长线持有和交易性机会。高位票是交易性机会不能长线不动；低位有大坑的票可以长线（如小V跌回50%但涨回20倍）。
- **案例**：米儿从17入场，分批在66、61等位置减仓。东方财富先从101跌到18，再从18拉到1245（122倍）。

## 交易纪律与止损类比——增量更新

**增量更新（2025-07-12 反内卷与新逻辑强化）：**

### 止损的投篮命中率类比
- **根本理念**：止损就像篮球手的投篮失手。再好的球星也没有百发百中，詹姆斯、杜兰特、库里三分命中率50%以上就算当日最佳。
- **核心**：25投13中，那12个miss就是止损成本——浪费了全队进攻回合。对散户而言止损成本是账户浮亏。
- **不能停止出手**：B1战法就是你最有效的进攻模式，十次五六次错，但三到四次做对就通过盈亏比回来了。
- **盈亏平衡点原则**：如果买进后从盈到亏，到盈亏平衡点必须止损出来。即使盘中来不及出来尾盘最多亏5%。

### 不跟盘原则（深化）
- 外面有各种炒股大赛跟单，但没有一个人能跟满。
- 原因：你只知道对方怎么做，不知道对方为什么这么做。节奏一步错步步错。
- 真正要解决的是知其然还要知其所以然。

## 分歧与一致（量价关系）——新术语

- **定义**：缩量横住后突然放量下跌（除非放量跌停），一般形成短期分歧。分歧之后又会走向一致。
- **市场意义**：一致后找缩量点，如果缩量点同时也是B1，则为极好的右侧买点（超级B1起点）。
- **特征**：分歧意味着多空双方意见不一。分歧之后如果缩量横住，说明市场重新达成一致（多空平衡），此时配合B1信号可入场。
