# 交易术语库：2025-05-07 永不套牢法

## 永不套牢法（少妇战法）
- **定义**：一套以"只输一根K线"为核心、永不套牢的中短线交易体系
- **核心规则**：
  1. **只输一根K线**：买入后以进场K线最低价下方设止损，破了就走
  2. **两根阳线减仓一半**：脱离BBI线后出现两根阳线，减仓一半（放飞/卤煮）
- **适用场景**：100万以下小资金；震荡行情尤其适合
- **市场意义**：积小胜为大胜，不被套就有持续参与市场的资格

## 只输一根K线（指数一根策略）
- **定义**：止损设在进场K线最低点下方几个价位（或1%幅度）
- **执行**：收盘跌破就走，不纠结、不幻想
- **市场意义**：永不套牢的核心风控手段
- **来源**：课程原文 "只输这一根K线……记住就输一根K线"

## BBI两根阳线减仓法（卤煮 / 放飞）
- **定义**：当股价向上脱离BBI线出现连续两根阳线时，必须减仓一半
- **目的**：降低成本，使剩余仓位拥有绝对成本优势
- **执行**：减仓一半后，剩余半仓盯BBI线持有
- **来源**：课程原文 "BBI线向上两根阳线一定要减仓一半，把利润给我留下来"

## 积小胜为大胜
- **定义**：通过多次小幅盈利，最终累积成可观收益
- **特征**：每次交易赚5%-10%，亏损控制在1%-5%
- **市场意义**：少妇战法的核心理念——"一路小屁走向胜利"

## 顺大势逆小势
- **定义**：在上涨大趋势中，等回调到B1位置时入场；在下跌趋势中不做多
- **执行**：识别大趋势方向 → 小趋势回调到支撑位/B1 → 入场
- **市场意义**：提高了胜率的交易哲学

## 大爷图 / 大野图 / 大业图
- **定义**：连续下跌趋势后的横盘整理图形
- **特征**：BBI向下，重心向下，看似有B1但不能碰
- **风险**：横盘末期容易向下破位加速下跌
- **规则**：**不要在大爷图上炫技**，宁可踏空不要做

## 大爷踏空图
- **定义**：上涨趋势中的横盘整理，回调到区间下沿的B1位置
- **特征**：前面有一波明确上涨 → 横盘回调 → 低点抬高
- **市场意义**：这种图形的B1胜率远高于下跌趋势横盘

## 关键K
- **定义**：决定趋势方向的关键K线，突破前高后成为后续持仓的依据
- **特征**：放量突破关键位置，之后该K线的最低价成为支撑位
- **市场意义**：关键K不破 = 趋势延续，可继续持有

## 金叉空（MACD金叉做空信号）
- **定义**：MACD顶背离 + 金叉 = 做空信号（期货/反向思维）
- **特征**：价格创新高但MACD红柱缩短或变绿，双线位置低于前高
- **市场意义**：在期货市场中用来做空的顶级信号

## 横盘区间画法
- **画法**：阶段最高点连线 = 上沿，阶段最低点连线 = 下沿
- **用法**：B1在下沿附近入场，上沿附近减仓/止盈
- **规则**：区间中间（可上可下）不碰，没有盈亏比

## 盈亏比原则
- **定义**：交易前先算向上空间和向下空间的比例
- **标准**：盈亏比至少3:1才值得做

## 战略相持
- **定义**：中美贸易战的第二阶段，双方互不相让
- **市场意义**：五月六月风险大，市场不会有流畅行情


## 本课系统贡献

**分析依据贡献：**
- 永不套牢法（少妇战法）完整交易体系：只输一根K线 + BBI两根阳线减仓一半
- 顺大势逆小势的交易哲学
- 上涨趋势横盘 vs 下跌趋势横盘的区别判断框架
- 盈亏比原则（至少3:1才值得做）

**分析依据贡献：**
- 只输一根K线止损策略（进场K线最低价下方1%）
- BBI两根阳线减仓一半策略（卤煮/放飞）
- 横盘区间画法（上沿压力/下沿支撑）
- 关键K线识别与运用
- MACD金叉空（做空信号）识别
- 左侧交易 vs 右侧交易的适用场景区分

**认知与纪律贡献：**
- 积小胜为大胜 → 不追求暴利，追求稳定复利
- 永不套牢 → 不被套就有持续参与市场的资格
- 不要炫技 → 下跌趋势横盘不要碰（大爷图法则）
- 遵守交易纪律 → 严格执行规则，不要因为前几次赚了就放松
- 踏空比亏钱更难受 → 所以永远要有一定的仓位
- 顺大势逆小势 → 做上涨趋势的回调，不做下跌趋势的反弹
