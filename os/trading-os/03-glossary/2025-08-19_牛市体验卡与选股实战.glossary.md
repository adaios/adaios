# 2025-08-19 牛市体验卡与选股实战

> 老师说这节课"特别贵"，属于 level 3 高阶课。

## 三最原则（核心框架）

完整的交易链条三环节：

```
选股：只选最美 → 盘中：只买最强 → 持仓：只拿最硬
```

### 只选最美

选股时只选完美图形，宁缺毋滥。

- **权重**：选股只占整个交易的20%（最重要是纪律50% + 盘面30%）
- **标准**：杨幂、迪丽热巴、古力娜扎、佟丽娅级别的选美标准，不要挖刘晓庆/马思纯
- **只选完美图形**：异动完美、N型完美、量价配合完美的图形
- **排除**：所有不完美的图形都排掉，换一批
- **原文金句：**"市场已经告诉你什么是顶美了，你就别那什么"

### 只买最强

盘中开盘后，基于实时量比和量价配合，只买自选池里最强的那个。

- **不预先期待**：不要对前一天选出的票抱有不切实际的幻想
- **盘中不幻想**：开盘量比不行就是不行，不要等涨停
- **只跟随不猜测**——这六个字是灵魂
- **权重**：盘中的决策占30%

### 只拿最硬

买入后，持仓里只留最强硬的票，不行的拍掉换下一批。

- **不留恋**：不要觉得"它不一样"，都一样
- **不给他第二次机会**：能套你一次就能套你两次
- **物来则应，过去不留**：涨了按规则放飞，跌了按规则止损
- **原文金句：**"好男儿志在四方，公主们全在包厢，不行就换下一批"

## 交易六大人性问题

老师总结的交易中六种常见心态，其中**五个是事实，一个是错觉**：

1. **一追就跌，一卖就涨** → 事实。原因是对N型结构和相对高低位置没概念
2. **死拿不涨，一卖就涨** → 事实。洗的就是你这类的对手盘，筹码不交不拉
3. **多选一必选错** → 事实。你选的那个一定是你看得懂的，但别人也看得懂
4. **原本想买没买涨了，买了的跌了** → ❌ **这是错觉，不是事实**。认知不够，真看好你早就找钱干进去了
5. **不敢买的涨，买的不涨** → 事实。**强者恒强**，市场永远是强者恒强
6. **重仓不涨，轻仓涨** → 事实。好东西都稀缺，能让你舒服买到的东西不值钱

## 量比开盘决策体系（核心工具）

基于通达信默认量比数值的完整开盘决策树：

### 低量比开盘（10以下）
- **今日大概率震荡**，可以等回调慢慢买入（买全天）
- 有一种例外：**强庄股或在下午随量比同步拉升**
- B1的量比：6-10就挺好，7以上就很好了

### 中量比开盘（7-16，对应东财5-14）
- **不显著，可上可下**
- 量价同步上涨 → 买方>卖方，今日单边上涨
- 量价不同步 → 观察

### 高量比开盘（20以上）+ 股价同步上涨
- **今日大概率上攻**，真金白银买出来的
- 开盘后股价上涨，量比保持高量比 → 真涨，干
- **原文金句：**"钱不会说谎"

### 高量比开盘 + 股价大幅下跌
- **大概率有问题，稳一手**
- 除非带量冲破日内高点，否则大概率单边下跌或转水下震荡
- 这是砸盘/假启动

### 中高量比（10以上）+ 大幅低开（2%+）
- 有人开盘就要走，**别接飞刀**
- 除非三波下杀结束后尾盘建仓

## 量比的执行流程

```
第一步：B1/B2选股 → 精选5-10个到自选池（只选最美）
第二步：第二天开盘 → 自选池内按量比排序（只买最强）
第三步：根据量比+量价配合决定干哪个
第四步：买入后切换为只拿最硬模式
```

- **关键**：量比不能在市场5000个票里瞎排序，必须在自己精选池里排序
- B2的量比天然比B1大（因为已经放过量），不能拿B1的标准要求B2

## 强者恒强

市场永远是强者恒强的逻辑。

- 不敢买的会一直涨，涨到所有人都相信为止
- 买的多的不涨因为不稀缺，买的少的涨因为稀缺
- **要做冠军粉**——不要跟钱过意不去
- 同理，同板块内选最早上攻的、涨得最快的、涨得最多的

---

## 配置是最好的风控

"不行就全买"不是真的全买，而是**配置思维**。

- **给一个股票十次机会，不如给十个股票一次机会**
- 大盘上涨时容错率高，配置把仓位推上去就行
- 大盘下跌时要精选个股
- 单吊一个板块等于没配置
- **原文金句：**"你养一条鱼死了会很难过，但养一池子鱼哪条死了你都不知道"

## 量比加分的突破战法

### 低位板规则
- B1/底部起来的第一个涨停 = **低位板**，可以干
- 高位涨停 = 不干
- B1的板就是低位，不需要再过脑子

### 事出反常必有妖
- 主升浪中出现奇怪的K线（放量阴线/螺旋桨/接近跌停）→ 拍掉
- 走流畅的票不会出现这种东西
- **原文金句：**"任何上涨主升浪的过程中出现丑的了，拍掉"

### 校准物方法
- 将成功案例的K线图截图收集（历史50张→逐步到100张）
- 每天选股之前先过一遍校准手感
- 类似品酒时用飞天53度口感校准
