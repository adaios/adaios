# 交易术语库：2025-04-23 大富翁导论

## B1（买点1）
- **定义**：N型下跌结构底部转折处的第一个买入点
- **特征**：连续下跌后出现反转K线信号
- **指标参考**：KDJ.J<20 左右（课程建议 KDJ.J<13 左右），J值到达大负值区间（即"KDJ大幅值"，指J值达到大的负值），数值只是参考
- **配合**：缩量确认 + 反转K线形态
- **市场意义**：技术面择时的入场信号
- **常见误区**：B1不是抄底，是确认转折后才介入

## 量化对冲
- **定义**：用数学模型捕捉市场机会 + 用对冲工具保护风险敞口
- **特征**：量化决定攻击性，对冲解决防守
- **市场意义**：防守反击制胜
- **大A现状**：国家队控盘后传统对冲策略失效，低波红利成为替代性"现货对冲"

## 4231阵型
- **定义**：仓位配置策略：40%低波红利（后卫防守）+ 30%前锋进攻 + 20%中场过渡 + 10%机动备用
- **特征**：每个持仓有明确的战术角色
- **市场意义**：通过配置实现天然对冲，降低组合波动
- **例外**：2000万以上资金可适当集中（"2000万反而不用做配置了，就单单抡一个"）

## 顶部大风车
- **定义**：连续大幅上涨后的见顶形态：放天量 + 长上影 + 长下影 + 螺旋桨
- **特征**：
  - 特指一根K线，不是多根K线组合
  - 大幅上涨后，主力已翻倍
  - 成交量天量
  - 上下影线 + 螺旋桨（像风车叶片）
- **市场意义**：出现该形态必须减仓，"走错也要走"
- **近似形态**：90%顶部大风车（上下影线稍短但仍可识别）
- **常见误区**：不是所有螺旋桨都是大风车，必须严格符合以上特征

## 3/4阴量线
- **定义**：成交量萎缩到前一天的75%左右的阴线
- **特征**：缩量但不是缩半量
- **市场意义**：不好的信号，尤其在突破位置出现
- **对比**：
  - **3/4阴量线（75%）：** 不好信号，大概率不会快速上涨
  - **缩半量（50%）：** 好信号，是最健康的洗盘形态
- **扩展**：阳量出现3/4也不理想，最好放量或平量

## 四不原则
- **定义**：不追、不动、不慌、不乱摸
- **特征**：系统化风控的口诀，课程将复杂风控系统"粗鄙地总结"成的口诀
- **市场意义**：应对市场波动的核心纪律
- **细化**：
  - **不追**：不追高，只在B1位置低吸
  - **不动**：核心资产不动，拿住不折腾
  - **不慌**：趋势向上、个股图形没坏，不因盘中波动恐慌卖出
  - **不乱摸**：不瞎操作，不炫技，不碰看不懂的票

## 专精图策略（二进制策略）
- **定义**：将K线阴阳转为二进制（阳=1，阴=0），通过历史回测找高概率模式
- **特征**：周期越长确定性越大；可做日线/周线级别，也可做分钟级别
- **实现**：需要编程做回测 + 硬件支持（如5090显卡），量化通过程序化自动执行
- **市场意义**：程序化交易的底层逻辑，赚"辛苦钱"

## 暴力特噗（暴力突破 / Trend Up）
- **定义**：放量大阳线突破前高，解放所有套牢盘后的突破形态
- **特征**：涨停或大阳线突破，成交量显著放大，彻底突破前期平台
- **配合**：后续不能出现3/4阴量线；最好放量或平量延续
- **市场意义**：暴力特噗 = 解放套牢盘 = 主力真金白银的信号

## 空仓也是策略
- **定义**：空仓是主动选择的策略状态，不是"没玩"
- **市场意义**：在判断风险大于收益时，空仓就是正确策略
- **说明**：宽松周期 + 经济拐点阶段空仓反而是错的（"菜鸟不能空仓"情况下空仓 = 择时方向判断错误）

## 每日压力测试
- **定义**：每个持仓自问连续两个跌停是否影响正常生活
- **公式**：极限仓位 = 能接受的亏损额 ÷ 跌停百分比
- **市场意义**：倒推出该票的极限仓位比例，保证睡眠质量

## 底部区间
- **定义**：底部是一个价格区间而不是一个点
- **画法**：高点连线 + 低点连线构成区间
- **市场意义**：在底部区间分批建仓，而非抄一个精确最低点
- **补充**：底部时间越久，机构拿货越充分，后续行情越可观

## 单顶不是顶
- **定义**：大盘股一天出不完货，需要二次冲高才能出货完毕
- **特征**：第一卖点（天量高位）+ 第二卖点（挑前高，MACD顶背离）
- **市场意义**：第一次见顶后可能有反弹/二次冲高，这是第二次出场机会

## 红肥绿瘦
- **定义**：阳线成交量明显大于阴线成交量
- **特征**：健康的建仓/持仓量价关系
- **市场意义**：机构真金白银买入的信号

## 择时三要素
- **定义**：基本面择时 + 资金面择时 + 技术面择时
- **基本面择时**：看宏观经济周期、政策方向（宽松周期判断）
- **资金面择时**：看活跃市值、Level2数据、增量资金进场信号（国家队动向）
- **技术面择时**：看K线形态、MACD（顶底背离+线上金叉）、成交量
- **顺序**：先基本面 → 再资金面 → 最后技术面

## 四个量化策略组合
- **热点概念跟踪组合**：热门行业里的核心龙头，形成组合
- **压抑拐点组合**：连续下跌到B1，拐点入场（越压抑起的越凶）
- **爆发追击组合**：牛市行情中的突破追入（目前尚未遇到真正的爆发）
- **小市值重组组合**：2015年前有效，国家队入驻后失效

## 专精图（专精图反转组合）
- **定义**：用二进制K线模型做历史回测，找高概率反转点
- **特征**：周线级别专精图确定性强；如连续阳线后接阴线，后面大概率继续阳

## 国家队资金动向
- **定义**：2015年股灾后国家队入场控盘，改变了市场生态
- **特征**：国家队入场十年，市场已被深度控盘
- **影响**：短线策略胜率降低；跟国家队配置方向（核心资产、低波红利）成为更安全的策略
- **指标**：低波红利涨 → 说明国家队/机构在避险 → 赚钱效应不高
- **信号**：拉板在上午10:30前封死 + 一字无量 → 主力控盘强；盘中反复打开 + 巨量 → 筹码松散

## MFI（资金流向指数）
- **定义**：课程提到的自用指标，基于Level2数据算出的机构控盘成本线和仓位变化
- **说明**：该指标为课程老师自用，非公开指标，用于判断机构是否真实建仓

**买入层贡献：**
- 4231阵型（仓位配置框架）→ 40%低波红利防守 + 30%前锋 + 20%中场 + 10%机动
- 四不原则（风控框架）→ 不追、不动、不慌、不乱摸

- 三大指标组合：MACD（顶底背离+线上金叉判断趋势）→ KDJ（大负值判断B1位置）→ 成交量/活跃市值（资金面择时）

- 3/4阴量线（突破后量价判断，不好信号）

**认知与纪律贡献：**
