# 2025-10-04 双线战法（国庆课程）

> 少妇战法的下一代，从一维（单线）升级为二维（双线），后面还有三维四维。

## 双线战法（核心）

从单线（EMA双重均线）升级为双线的交易体系。

**白线（知行短期趋势线/牵牛绳）**：
- 公式：EMA(EMA(C,10),10)，双重指数移动平均
- **不是BBI**，但用法跟BBI类似
- 股价短期快速上涨期间一般不会跌破白线
- 建仓期/拉升期/冲刺期流畅上涨都在白线上
- 破白线 = 短期趋势大概率走坏

**黄线（知行多空线/大哥线/主力成本线）**：
- 公式：(MA(14)+MA(28)+MA(57)+MA(114))/4，四周期均线平均
- **不是MA60**，虽然接近
- 主力是否入场的标志性参考线
- 极限洗盘的极限位（洗到黄线你最害怕的时候就是底）
- 大哥控盘的成本线
- 股价在黄线之下 = 没有交易价值，都是反弹
- 黄线之下 + 白线之下 = 不看不做

## 双线核心用法（4条）

### 1. 右侧只做白线在黄线上的票
- 股价在黄线之下的都不看，浪费时间
- 白线金叉黄线 → 上涨趋势确立的概率大
- 拿着到死叉就是"大傻子玩法"——能吃满一个波段，但掐头去尾

### 2. 短期放量站上白线才有上涨爆发力
- 只有放量站上白线才具备拉升形态
- 连续拉升后回踩白线 = 第一支撑
- 破白线 = 短期趋势走坏，全清

### 3. 白线与黄线要死叉而未死叉 + N型结构不破 + 勾大幅值 = 极限买点
- 死叉多信号
- 越是大票（流动性强）越有效
- 成交量越大，技术指标越有效（因为画不了线）

### 4. **N型上涨趋势中，白线与黄线之间都是容错率较高的买入区** ← 最核心
- 白线金叉黄线后，趋势已形成
- N型结构不破的前提下，白线和黄线之间的任何位置都可以买
- 无非是买多买少的问题
- 这是一种"模糊的正确"
- 止损设黄线
- 大赚小亏 + 配置 = 稳定盈利

## 死叉 = 最后离场时机
- 白线死叉黄线 = 主力弃盘
- 走错也要走
- 青花案例：205死叉，不走跌到164

## 本课系统贡献

**框架层：**
- **双线战法**：从少妇战法（单线）升级到双线体系
- 白线（短期趋势线）+ 黄线（大哥线/主力成本线）的完整框架
- N型上涨趋势中白黄线区间都是买入区（最核心）

**工具层：**
- 白线金叉黄线 = 上涨趋势确立
- 白线死叉黄线 = 最后离场时机
- 死叉未死叉 + N型 + 勾大幅值 = 极限买点

**认知层：**
- 模糊的正确优于精确的错误
- 技术指标是人性的量化
- 成交量越大的票，技术指标越有效
