## 交易体系总纲

```
①择时 → ②选股 → ③等B1 → ④设止损 → ⑤止盈放飞 → ⑥收队
```

**执行口诀：**
```
B1买 → 设止损 → 破了平 → 涨起来白线（知行趋势线）上两根中阳线放飞一半 → 看白线（知行趋势线）→ 跌破两根全离场
```

**两条路线：**
- **底线**：少妇战法六步，靠交易赚波段的钱
- **上限**：主仓拿住不动（赚大钱）+ 副仓少妇战法（做增强），两条腿走路

**三最原则（高阶交易框架）：**
```
只选最美（选股）→ 只买最强（盘中）→ 只拿最硬（持仓）
```

**各环节权重分配：** 交易纪律50% + 盘面30% + 选股20%

**盈利模型：** 每年250个交易日，用40个止损B1换5个大牛B1。执行这个策略就能稳定盈利。

## 第一步│择时（什么时候做）

### 1.1 底层认知（理解 A 股的底层逻辑）

```json
{
  "A股底层六点": [
    "长期充分博弈 → 各方参与者极致内卷",
    "交易机制不对等 → 机构T+0 vs 散户T+1",
    "老麒麟会根深蒂固 → 每个老票背后都有主",
    "洋务派主导大局 → 留学背景基金经理看美股脸色",
    "政策驱动机制 → 国家队和制度安排引导方向",
    "独特盈利模式 → 不需要单边牛市，只需波动中获利"
  ],
  "扩展认知": [
    "财政与股权绑定：龙队的钱不能输，上有顶下有底",
    "A股本质是货币现象：存款搬家才是见顶信号",
    "大A不会踏空：长期参与者不必因一两天上涨而焦虑",
    "顺周期启动顺序：银行→券商→科技→保险→消费"
  ]
}
```

### 1.2 择时三要素

| 要素 | 关注什么 | 核心工具 |
|:----|:---------|:---------|
| 基本面择时 | 宏观周期、经济拐点 | 双宽政策、经济数据 |
| 资金面择时（最重要） | 增量资金入场 | 活跃市值、Level2、B1数量 |
| 技术面择时 | 具体买点 | K线形态、MACD、成交量 |

### 1.3 活跃市值信号速查

**启动信号：**
- 单日 >= 4% 或连续两日累计 >= 4% = 新波段开始
- 5%-8% 阳线 = 主线级别；7%-20% = 全市场普涨

**离场信号：**
- 单日跌幅 >= 2.3% = 大概率波段见顶，转为"只卖不买"
- 连续数日重心下移 = 不是猛干的时候
- 跳空低开放量下杀 = 阶段性底部临近

**区间切换：**

| 区间 | 特征 | 操作 |
|:----|:-----|:-----|
| 多头区间 | 活跃市值红柱上升 | 容错率高，B1随便做，大胆做 |
| 空头区间 | 活跃市值绿柱下降 | 收紧策略，不出手等信号 |
| 切换确认 | 零轴之上二次金叉 | 转多头思维 |

**深层逻辑：**
- 活跃市值本质是**筹码与钱的关系**，不是简单的资金流入流出
- **筹码沉淀**：大量长线资金锁仓是龙队控指数的基础
- **浮筹清洗**：新行情启动前必须把之前参与者洗走
- **强势股补跌**是新行情前奏，早涨早跌早反弹

**原则：**
- 宽松周期+经济拐点：**不能空仓**（至少半仓）
- 判断风险>收益：**空仓也是交易策略**
- 策略必须分**上涨/下跌/横盘**三区间回测
- **时间换空间**：阴跌磨底消化浮筹，跟着磨

### 1.4 MACD 三种用法

| 用法 | 定义 | 实战意义 |
|:----|:-----|:---------|
| **零轴多空** | 快线（白线）在零轴以上=多头，以下=空头 | 先看线再看柱。线是趋势，柱可对倒 |
| **顶底背离** | 价格新高/低但白线没跟上 | 底背离看周线更可靠，顶背离看日线 |
| **金叉空/死叉多** | 金叉前拐头向下/死叉前拐头向上 | 后者=空中加油。金叉空位置的滴滴最有参考价值 |

**本质：** 顺周期=建仓波（真买）=>洗盘=>筹码锁定=>不花钱也能拉。完整周期：建仓=>洗盘=>初升=>洗盘=>主升=>出货=>MACD变绿=>空头区间。

**MACD探底法：** MACD从零轴上方穿到下方绿柱，走到接近前低极值区域（如-4.8、-5.x）=波段底部区域。

## 第二步│选股（做什么票）

### 2.1 选股逻辑链

```
基本面看政策 → 政策看行业 → 行业看龙头 → 盘面语言确认
```

### 2.2 核心原则

| 原则 | 表述 |
|:----|:-----|
| **不离开主线** | 每个阶段市场有明确主线，偏离容易踏空 |
| **不碰下跌趋势** | 大野图（地产等全线周线空头）不碰 |
| **选不出来=进步** | 无聊行情选不出B1说明审美提高了，宁缺毋滥 |
| **板块龙头验证** | 板块要有行情，大盘龙头必须先翻番 |
| **不研究一波流票** | 一气呵成不给上车机会的，换一批 |

### 2.3 配置框架

**主仓+副仓分离：**
- **主仓（左仓）**：确定性强的票拿去不动，睡得好觉
- **副仓（右仓）**：少妇战法做交易增强，赚波段钱
- 不能把副仓拿成主仓，也不能把主仓在低位拍掉

**曼城阵容（大资金）：** 4231阵型：40%低波红利防守+30%前锋进攻+20%中场过渡+10%机动

**基石+前锋：** 60%基石防守 + 40%前锋进攻，从防守转进攻要有明确切换逻辑

**池子结构（主线/支线/答应）：**

| 层级 | 方向 | 配置角色 |
|:----|:-----|:---------|
| 趋势中枢 | 半导体（六大攻关） | 长线底仓 |
| 落地方向 | AIGC应用（十五五落地） | 进攻前锋 |
| 核心防守 | 白酒/煤炭 | 基石防守 |
| 辅助方向 | 创新药、储能、机器人 | 波段增强 |

### 2.4 ETF 选行业

- 选偏软件不偏硬件，偏应用不偏游戏，偏AIGC不偏算力
- ETF入场也用B1体系（周线B1建仓法：小仓位打底=>往下漏再加=>异动回踩推满）

## 第三步│入场（怎么买）

### 3.1 B1（核心入场方式）

```
条件：连续下跌 → N型转折 → KDJ大负值（J<13） → 缩量确认
```

- B1不是抄底，是确认转折后才介入
- 配合MACD底背离+白线（知行趋势线）向上，确定性更强
- 一般第二天开盘买入

**递进确认链：** B1 => 再B1 => SB1，不要一个B1就满仓，等后续确认再加仓
**B1要爆发力：** 脱离成本区要快，缩量B1可靠性低于放量B1
**B1三天原则：** 3天内起来就起来了，第4天开始概率衰减

### 3.2 买点优劣对比

| 买点 | 盈亏比 | 确定性 | 适合谁 |
|:----|:------|:-------|:-------|
| B1 | 最高 | 中 | 所有人 |
| B2 | 高 | 高 | 有经验者 |
| B3 | 中 | 最高 | 高手（99人不做）|
| SB1 | 中 | 很高 | 忍出来的买点 |

**B2止损：** 第二天K线低点破掉就走（滴滴规则）
**B3：** B2之后的大阳线或缩量阳小十字星。B3只看白线（知行趋势线），配合砖形图的就是大牛

### 3.3 特殊入场方式

| 方式 | 条件 | 注意 |
|:----|:-----|:-----|
| **暴力特噗** | 放量大阳线突破前高 | 不能出现3/4阴量线 |
| **深水炸弹** | 连续大跌+ADX拐头向上 | 3天不涨必走 |
| **单针下20/30** | 知行洗盘副图白线被打到20或30以下 | 数值越小洗得越狠 |
| **补票战法** | B1错过后的补票方案 | 突破后入场 |
| **底部暴力K线** | 小暴力（趋势延续）最具操作价值 | 四条选股原则：底部+突兀+粗大+让人害怕 |

### 3.4 底部暴力K线体系

| 类型 | 定义 | 交易意义 |
|:----|:-----|:---------|
| 大暴力-趋势扭转 | 长期下跌后第一根放量大阳 | 只用作观察 |
| 大暴力-趋势改变 | 横盘后改变方向的放量大阳 | 可操作，配黄线站上+关键K确认 |
| **小暴力-趋势延续** | 上涨中回调到第一位置 | **最具操作价值** |

**四条选股原则：** 底部位置 + 突兀/莫名其妙 + 又粗又大 + 让人害怕的时间点

### 3.5 买入三问

每条B1操作前必答：
1. 买入预期是什么？（到前高？突破？填坑？）
2. 盈亏比划不划算？（短线至少3:1）
3. 哪里设止损？（前低/N型最低点/入场K线最低价下方）

### 3.6 洗盘识别

- 股价没怎么跌，KDJ快速下到大负值 = 洗盘不是出货 => B1买入良机
- **击穿对手盘**：主力故意跌破黄线，限跟风大资金筹码洗出
- **假破位识别**：破坑沿位不破低位，是主力洗盘手法

### 3.7 假突破识别

> 大A市场 **90%以上的突破是假突破**，量化专门反打突破。
> **应对：** 别做突破，买B1。在N型结构低点买B1比追突破舒服得多。

## 第四步│设止损（怎么防）

### 4.1 核心理念：只输一根K线

```
止损设在进场K线最低价下方几个价位（或1%），收盘跌破就走。永不套牢。
```

### 4.2 止损三级别

| 级别 | 设置方法 | 适用 |
|:----|:---------|:-----|
| 敏感型 | 进场K线最低价下方3-5个价位 | 新手/小资金 |
| 趋势型 | 本阶段最低点或上个阶段最低点 | 有经验者 |
| 盈亏比型 | 按盈亏比算 | 大资金 |

### 4.3 止损三原则

1. **赚钱的票不要做亏**：涨起来又跌回成本价，拍掉
2. **买入后该涨不涨拍掉**：成本区附近磨叽，拍掉（短线三天不涨拍掉）
3. **破掉止损价格拍掉**：收盘前确认，不盘中拍

### 4.4 多层防守体系

```
重要性从高到低：没涨 → 盈转亏 → 止损 → 放飞 → S1 → 滴滴（阶梯量）→ 破白线（知行趋势线）→ 破黄线 → 死叉
```

| 防守层 | 信号 | 操作 |
|:-------|:-----|:------|
| **没涨**（最高优先级） | 买入后未脱离成本区 | 拍掉换下一批 |
| **盈转亏** | 盘中浮盈转浮亏 | 不等收盘，直接拍掉 |
| **S1** | 加速后放巨量大阴线/大风车 | 大票卖1/3-1/2，小票全卖 |
| **滴滴（阶梯量）** | 连续多根阴线逐级缩量 | 需要足够时间消化，等SB1 |
| **破白线（知行趋势线）** | 短期趋势走坏 | 白线破掉看黄线 |
| **破黄线** | 高位放量跌破 | 最后的防守线，必须走 |
| **死叉** | 白线死叉黄线 | 最后离场时机，走错也要走 |

### 4.5 关键止损规则

- **假阴真阳**：一律当真看，等突破确认
- **金叉空位置的滴滴**：最有参考价值，协和医院级别的多空确认
- **跌破黄线第二天确认再走**：可以不教条，取决于承受力
- **缩量突破=控盘**：筹码都在主力手里，后续看涨
- **每日压力测试**：连续两个跌停不影响睡觉=最大仓位

## 第五步│止盈放飞（怎么减）

### 5.1 核心规则

```
白线（知行趋势线）上出现两根中阳线 → 减仓一半（卤煮/放飞）
```

剩余半仓以白线（知行趋势线）为持股依据。

### 5.2 中阳线标准（因票而异）

| 股票类型 | 中阳线标准 |
|:---------|:-----------|
| 低波动票 | 3% |
| 券商 | 4%-6% |
| 白酒 | 3.5%-6% |
| 创业板20cm | 7%以上 |

### 5.3 减仓节奏

- 第一波减半后继续大涨可再减至1/4
- 涨停板放飞原则：B1入场的涨停板放不放飞都可以，二次封板坚决可以不飞
- 第四块砖放飞铁律：砖形图出现第四块红砖时必须考虑放飞
- DSZ战法（1234战法）：连续四连涨后收绿=高概率调整，第8根K线考虑减仓
- 主升中敢于放飞：浮盈多的时候不要该飞不飞

### 5.4 见顶信号

| 信号 | 识别 | 操作 |
|:----|:-----|:-----|
| **顶部大风车** | 放天量+长上影+长下影+螺旋桨（单根K线） | 走错也要走 |
| **S1/S2/S3** | 加速放量=>双头顶背离=>放量下沿反弹确认 | 全套逃顶体系 |
| **MACD顶背离** | 价格新高但红柱缩短/白线走低 | 只在放巨量之后有效 |
| **单顶不是顶** | 大盘股一天出不完货 | 有二次冲高机会 |
| **五天换手率出货** | 5日换手100%+，量不突增但持续大 | 不等放天量，已出完 |

## 第六步│收队（怎么走）

### 6.1 核心规则

```
跌破白线（知行趋势线）两根K线（收盘价）→ 全仓离场
```

不以收盘价为准。不和股票谈恋爱。

### 6.2 出场原则

| 场景 | 操作 |
|:----|:-----|
| 顶部形态 | 宁可信其有，走错也要走 |
| 单顶不是顶 | 大盘股有二次冲高机会 |
| 下跌中的阳线叫反弹 | 到压力位必须出，不是反转 |
| 行情末期 | 不做第二次入场 |

### 6.3 收队不是清空

收队是减仓到防守仓位，不是全部清掉。空头区间不要干，等下一波信号。

### 6.4 孤注一掷不可怕

如果一笔交易输了就伤筋动骨，就不要做。

## 附录A：K线形态速查

### A1 单根K线信号

| 形态 | 信号 |
|:----|:-----|
| 长下影阳线 | 多方完爆空方，继续看涨 |
| 长上影阳线 | 试探遇阻，守实体低点可看涨 |
| 长下影阴线 | 空头遇抵抗，不破实体高点继续看跌 |
| 长上影阴线 | 空头完爆多方，破低看空 |
| 十字星（趋势末端） | 反转概率高 |
| 标准K线 | 趋势延续 |

### A2 量价信号

| 信号 | 含义 |
|:----|:-----|
| **倍量柱** | 成交量2倍及以上=有价值，上涨中重点关注 |
| **天量柱** | 倍量后再倍量=阶段性到顶 |
| **平量** | 成交量持平=趋势延续信号 |
| **一般放量** | 既非倍量也非天量，没有意义 |
| **阶梯量** | 出货信号，需时间消化 |
| **红肥绿瘦** | 阳量>阴量=多头主导 |
| **3/4阴量线** | 突破后缩量75%=不好信号 |
| **缩半量** | 洗盘健康信号 |

### A3 图形分类

| 图形 | 判断 | 操作 |
|:----|:-----|:-----|
| 大爷踏空图 | 上涨趋势横盘回调下沿 | 可做B1 |
| 大爷图/大野图 | 下跌趋势横盘、重心向下 | 不能碰 |
| N型上涨 | 低点抬高+高点抬高 | 健康趋势 |
| N型下跌 | 高点降低+低点降低 | 等B1 |
| 麒麟会犬牙图形 | 密集上下影线交织 | 不看不碰 |
| 走流畅票 | 沿白线（知行趋势线）缓步上行 | 防卖飞，拿住不动 |

### A4 砖形图（专精图）节奏

| 形态 | 含义 | 操作 |
|:----|:-----|:-----|
| 红砖连续+绿砖横盘+白线上来 | 强势洗盘（技术调整） | 新红砖时买入 |
| 第四块红砖以上 | 追高危险 | 等调整再入 |
| 绿砖缩量横盘后新红砖 | 洗盘结束 | 买入信号 |

## 附录B：仓位管理

| 规则 | 内容 |
|:----|:-----|
| **四不原则** | 不追 => 不动 => 不慌 => 不乱摸 |
| **分仓** | 100万以下分4-5仓 |
| **压力测试** | 连续两个跌停不影响睡觉=最大仓位 |

| **短线不加仓** | 没有脱离成本区前绝对不加仓 |
| **分散** | 与其给一个板块十次机会，不如给十个板块一次机会 |
| **全力追一个方向要小心见顶** | 当全市场都割掉白酒追科技时，科技就要到头了 |

### 核心认知清单（按重要程度排列）

1. **赚大钱 vs 赚小钱**：少妇战法赚交易钱，左仓赚认知钱，两条路不能混用
2. **杨幂理论**：持仓不涨但出现更完美图形时无条件切换
3. **习惯打败人性**：不需要克服人性，多做几次形成肌肉记忆
4. **心态是交易核心**：走势为打崩你心态而设计
5. **消息面都是噪音**：上涨是主力在干活，下跌是主力要走
6. **先不死原则**：第一原则不是赚钱，是保证不亏大钱
7. **交易应对比预测重要**：到了位置就应对，不基于假设操作
8. **不给第二次机会**：能套你一次就能套你两次
9. **强者恒强**：做冠军粉
10. **不以账户盈亏做决策**：基于盘面做决策
11. **散户做情绪，机构做逻辑**
12. **从小钱滚大靠波段，从大钱到自由靠长线**
13. **产业资本视角**：持有1万股就是资本，老板在给你打工
14. **投资者成长三阶段**：短线=>波段=>配置
15. **没看懂就不干**：天天空仓不亏钱
16. **高位三波不做**：一把回到解放前

### 双线战法

| 线 | 定义 | 核心用法 |
|:---|:-----|:---------|
| **白线（知行趋势线）** | EMA(EMA(C,10),10)，双重指数移动平均 | 短期趋势线。破白线=短期趋势走坏，全清 |
| **黄线（大哥线）** | (MA(14)+MA(28)+MA(57)+MA(114))/4 | 主力入场参考线。白线金叉黄线=上涨趋势确立 |

**核心用法：**
- 右侧只做白线在黄线上的票
- N型上涨中，白线与黄线之间是容错率最高的买入区
- 白线死叉黄线=最后离场时机，走错也要走
- 股价在黄线+白线之下=没有交易价值
