# Research

## 定位

持续研究影响未来 3~5 年中国资本市场的重要趋势。

不同于交易课程（01-10），本模块更关注宏观→行业→公司的分析链条。

## 文件索引

| # | 文件 | 角色 | 内容 | 更新频率 |
|:-:|:----|:----|:-----|:---------|
| 00 | `ipo-tracker.md` | 🚪 **入口** | IPO总表（层次→公司→十五五行业→资本→状态→看点） | 收到IPO消息时 |
| 01 | `fifteen-five.md` | 🏗️ **框架** | 十五五行业方向 + 诞生路径 + 判断标准 | 极少，认知升级时 |
| 02 | `capital-decode.md` | 🔍 **分析工具** | 各赛道资金性质 + 博弈逻辑 + 三重共振模型 | 有新的资本认知时 |
| 03 | `regional-capital.md` | 🗺️ **深耕** | 地方国资地图 + 典型案例（合肥模式等）| 有地方资本运作时 |
| 04 | `industries.md` | 🌊 **行业涟漪** | 行业级事件 + 逻辑变化积累 | IPO扩散到行业时 |
| 05 | `companies.md` | 📁 **公司档案** | 重点公司深度档案 | IPO扩散到公司时 |
| 06 | `inbox.md` | 📥 **收件箱** | 零散信息先记录，标记 ✅ 后等收敛 | 随时 |
| deep-dives/ | 📚 **深度研究** | 产业链深度分析（一篇一个公司） | 用户投喂产业链分析时 |

## 数据流

```
你投喂一条 IPO 消息
    ↓
00-ipo-tracker.md ← 第一站：总表 + 归入十五五行业 + 看背后资本 + 分析判断
    ↓
┌─ 如果要深入研究 ──────────────────────────────────┐
│ 04-industries（行业涟漪）                          │
│ 05-companies（公司档案）                           │
│ deep-dives/（产业链深度）                           │
│    ↓                                               │
│ 回溯到框架层找"为什么"：                           │
│ 01-fifteen-five（为什么归这个行业？）              │
│ 02-capital-decode（这个资本有什么特点？）           │
│ 03-regional-capital（背后是哪个地方国资？案例？）   │
└────────────────────────────────────────────────────┘
    ↓（如改变宏观认知）
01-fifteen-five / 02-capital-decode / 03-regional-capital 增量更新
```

## 1条消息 → 完整的"一石激起千层浪"

以宇树科技为例，一条IPO消息完整的扩散路径：

```
用户投喂（宇树招股书+产业链分析）
    ↓
00-ipo-tracker 更新总表+日志
    ↓
deep-dives/宇树科技.md  ← 完整深度分析
    ↓
04-industries 机器人★★★★★ + 产业链图谱
    ↓
05-companies 宇树公司档案
    ↓
01-fifteen-five 验证了IPO扩张→二级映射路径
    ↓
02-capital-decode 机器人三重共振状态确认
    ↓
03-regional-capital 深圳模式案例补充
```

目标不是记录新闻，而是建立长期认知。
最终形成：**宏观 → 行业 → 公司 → 交易** 的分析框架。
