## 三、API 方案详解

### 3.1 东方财富 API（免费，推荐优先实现）

东方财富有非官方的公开数据接口，无需 API Key，直接 HTTP 请求：

**个股行情/实时数据：**
```
https://push2.eastmoney.com/api/qt/stock/get?secid=1.688825&fields=f43,f44,f45,f46,f47,f48,f50,f57,f58,f170
```
参数：`secid=市场.代码`（1=上交所，0=深交所，0.300XXX=创业板）

**板块/概念实时行情：**
```
https://push2.eastmoney.com/api/qt/clist/get?pn=1&pz=50&fs=m:90+t:2&fields=f12,f14,f3,f4,f62
```
- `m:90+t:2` = 概念板块，`m:90+t:3` = 行业板块
- `f3` = 涨跌幅，`f4` = 涨跌额，`f62` = 主力净流入

**板块成分股：**
```
https://push2.eastmoney.com/api/qt/clist/get?pn=1&pz=100&fs=b:BKXXXX&fields=f12,f14,f3,f4,f6,f7,f15,f16,f17,f18
```

**新闻/资讯：**
```
https://search-api-web.eastmoney.com/search/jsonp?param=长鑫科技&type=8197&page=1
```

**优点：** 免费、无认证、数据实时、覆盖全市场
**缺点：** 非官方文档（但已广泛使用多年），ID/编码体系需要映射

### 3.2 财联社快讯（电报）⭐⭐⭐⭐⭐

财联社的快讯是 A 股最快速的消息源之一：

```
https://www.cls.cn/telegraph  →  Web抓取页面解析
```

财联社也有非官方 API：
```
https://www.cls.cn/api/sw?app=CailianpressWeb&os=web&sv=8.4.0
```

**优点：** 消息最早，覆盖面全，从宏观到行业到个股
**缺点：** 需要轮询频率高才有意义

### 3.3 同花顺 iFinD / 通达信（需要客户端）

| 方案 | 费用 | 能力 | 适用场景 |
|:----|:-----|:-----|:--------|
| **同花顺 iFinD API** | 付费（机构版数万/年） | 全市场数据+新闻+研报 | 专业级 |
| **通达信公式** | 免费（已有证券账户） | L2行情、板块数据 | 盘中使用 |
| **同花顺问财** | 免费 | 自然语言搜索 | 快速查询 |

对于你这个项目来说，**东方财富 API + 财联社抓取** 是最快能跑起来的组合，零成本。通达信/同花顺的客户端 API 属于锦上添花。

### 3.4 RSS 方案（轻量，但逐渐萎缩）

中国主流财经站基本已关闭 RSS 输出。可用的替代方案：

| 方案 | 说明 |
|:----|:-----|
| **各站 sitemap.xml** | 多数站有，可解析最新文章列表 |
| **即刻/竹白等 newsletter** | 被动接收，但信息密度低 |
| **RSSHub** | 自建 RSS 生成器，可把各大站转为 RSS |

**结论：RSS 在中国财经领域基本已死，不建议投入。直接抓 API + 页面解析更可靠。**

## 四、建议的实施优先级

| 优先级 | 做什么 | 效果 |
|:------|:------|:-----|
| **P0** | 加东方财富 API 脚本（搜索+行情+新闻） | 替代 search_web.py，根治中文分词问题 |
