# 交易规则：2025-04-23 大富翁导论

> **课程时间**：2025-04-23
> **课程主题**：大富翁导论——量化策略基础与交易入门
> **核心概念**：量化对冲、4231阵型、四不原则、择时三要素

---

## 一、风控规则

### 规则 1：四不原则
- **不追**：不追高，只在B1位置低吸
- **不动**：核心资产不动，拿住不折腾
- **不慌**：趋势向上，个股图形没坏，不因盘中波动恐慌卖出
- **不乱摸**：不瞎操作，不炫技，不碰看不懂的票
- **来源**：课程原文 "不追不动，不慌不乱摸"
- **备注**：这是课程将复杂风控系统"粗鄙地总结"成的口诀

### 规则 2：每日压力测试
- **内容**：每个持仓自问：如果它连续两个跌停，是否影响我正常生活？
- **执行**：如果影响，说明该票仓位过大，必须减仓
- **公式**：极限仓位 = 能接受的亏损金额 ÷ 单票连续跌停百分比
- **来源**：课程原文 "你拿一个票，他连续两个跌停都不影响你睡觉，那么说明那个票就是你能持有的最大仓位"

### 规则 3：用输得起的钱炒股
- **内容**：只能用输得起的钱参与市场，不能拿救命钱、买房钱、借来的钱
- **来源**：课程原文 "如果你要是100万，你就想如果真有一天100万变成90万了，我媳妇得抽我，他妈就不让我上床了"
- **备注**：仓位上限 = 能接受的单日最大亏损 ÷ 满仓跌停百分比

### 规则 4：止损纪律
- **内容**：买入前必须确定止损位，破了就走
- **设置方法**：
  - 放量大阳线的最低价往下放两三个价位
  - 或直接把前期低点设为止损
- **原则**：越完美的图形越要干，但前提是有止损
- **来源**：课程原文 "这根放量大阳的最低价设好止损，你往下可以两个价位……把前期低点设为止损"
- **备注**："越完美越要干，但是不是让你死扛"

## 二、仓位管理规则

### 规则 5：4231 仓位配置
- **结构**：
  - 40% 低波红利（后卫防守）— 银行、电力、红利ETF
  - 30% 前锋进攻 — 科技、成长龙头
  - 20% 中场 — 均衡配置
  - 10% 机动 — 现金备用
- **原则**：每个仓位有明确的战术角色，知道每个票是干嘛的
- **来源**：课程原文 "你平时现在仓里什么票，你自己一定要知道。你有五个票，你有三个票，你有两个票，你一定知道每个票是干嘛的"
- **备注**："40低波红利是防守，前锋是进攻，中场是过渡，机动是备用"

### 规则 6：必须配置，不单调
- **内容**：不管多少钱，1万也要配置，不能单调一只
- **原则**：重仓一只赚大钱的人，将来也会重仓一只亏大钱（盈亏同源）
- **来源**：课程原文 "你能单轮一次单轮一个票赚到大钱，你以后就百分百能单抡一个票套一个大个的"
- **例外**：2000万以上资金可适当集中（"2000万反而不用做配置了，就单单抡一个"）

### 规则 7：每个持仓必须有明确角色
- **内容**：持仓中的每个票要知道它是干吗用的
  - 银行/电力/红利低波 = 对冲用（不能期望它上涨）
  - 科技 = 前锋，大盘涨时必须涨；不涨就说明它不在热点上
- **来源**：课程原文 "你现在买银行绝逼是对冲用的，你整电力绝逼是对冲用的"
- **备注**：大盘涨时科技票不涨 → 它不是热点/龙头 → 拍掉

## 三、入场规则

### 规则 8：B1 买入
- **定义**：N型下跌结构底部转折处的第一个买入点
- **特征**：连续下跌后出现反转K线信号
- **依据**：配合KDJ大负值（KDJ.J<20，参考值KDJ.J<13，J值达到大的负值，即"KDJ大幅值"）+ 缩量确认 + 反转K线
- **来源**：课程原文 "连续跌三天之后买"

### 规则 9：暴力特噗买入（暴力突破）
- **定义**：放量大阳线突破前高，解放所有套牢盘后的买入机会
- **特征**：涨停或大阳线突破前高，成交量显著放大
- **配合**：后续不能出现3/4阴量线；最好放量或平量延续
- **来源**：课程原文 "暴力特噗就是这意思"
- **备注**：暴力特噗的后继形态中，3/4阴量线 = 不好的信号，大概率不会快速上涨

### 规则 10：干错也要干
- **内容**：当技术图形完美时，即使犹豫也必须按纪律执行买入
- **前提**：
  - 图形必须真正完美（底部放量、顶部无量、缩量回调到B1）
  - 必须设好止损
  - 干错的前提是有风控，不能把自己扎进去
- **来源**：课程原文 "我的交易纪律是越完美越要干，因为什么？干错也要干，但是不是让你死扛"

### 规则 11：拉板形态判断
- **内容**：真正的涨停封板在上午10:30前封死 + 当日一字无量 → 主力控盘强
- **反例**：盘中反复打开 + 巨量 → 筹码松散，不会连续上涨
- **来源**：课程原文 "真正的涨停，一定是在上午十点半之前，把封住之后，当天一字这块完全没有量"

## 四、出场规则

### 规则 12：顶部大风车减仓
- **定义**：连续大幅上涨后出现放天量 + 长上影 + 长下影 + 螺旋桨
- **特征**：特指一根K线，不是多根K线组合；主力已翻倍
- **执行**：出现该形态必须减仓，"走错也要走"
- **严格区分**：不是所有螺旋桨都是大风车
  - 必须大幅上涨后（主力成本翻倍以上）
  - 必须是天量（对比历史天量级别）
  - 上下影线 + 螺旋桨完整
- **来源**：课程原文 "顶部大风车一定是连续上涨了之后，明显已经完全脱离主力的成本……放巨量，加上影线下影线，中间这个螺旋桨"
- **备注**：出现顶部大风车后，99.9%概率不会再翻过去

### 规则 13：单顶不是顶，二次冲高是第二卖点
- **内容**：大盘股一天出不完货，第一次见顶后可能有二次冲高
- **两个卖点**：
  - 第一卖点：天量高位（如顶部大风车出现时）
  - 第二卖点：挑前高时MACD顶背离（红柱明显缩小或死叉）
- **来源**：课程原文 "单顶不是顶出不完，尤其这么大盘子出不完"
- **指标**：大趋势票登场用MACD，不要用KDJ

### 规则 14：N型止损
- **内容**：N型下跌结构中，把前一低点设为止损位
- **来源**：课程原文 "如果你这根大阳起来……把它的前期的这个低点，把这个低点最低价往下聊两三个价位当成止损"

## 五、择时规则

### 规则 15：择时三要素
- **基本面择时**：看宏观经济周期、政策方向（宽松周期判断）
- **资金面择时**：看活跃市值、Level2数据、增量资金进场信号
- **技术面择时**：看K线形态、MACD、成交量
- **顺序**：先判断大方向（基本面）→ 再看资金动向（资金面）→ 最后找具体买点（技术面）
- **来源**：课程原文 "先基本面，再资金面，再技术面"

### 规则 16：宽松周期不能空仓
- **内容**：宽松周期 + 经济从下往上走拐点，至少维持半仓
- **例外**：如果判断风险大于收益，空仓也是正确策略
- **来源**：课程原文 "从现在开始到924开始到现在，如果你要是空仓，那你就是菜鸟不能空仓"

### 规则 17：底部区间分批建仓
- **内容**：底部是价格区间不是一个点，在区间内分批买入
- **画法**：高点连线 + 低点连线构成底部区间
- **来源**：课程原文 "底部是个区间……高点连线+低点连线"

### 规则 18：低波红利走势判断市场强弱
- **内容**：低波红利涨 → 国家队/机构在避险 → 市场赚钱效应不高
- **来源**：课程原文 "当低波红利涨，那么说明市场里面赚钱效应并不高"

## 六、持仓规则

### 规则 19：红肥绿瘦是健康信号
- **内容**：阳线成交量明显大于阴线成交量 = 健康建仓/持仓
- **来源**：课程原文 "底下阳量比较大，同时又比阴量多，就是红肥绿瘦"

### 规则 20：跟国家队在一起
- **内容**：国家队入场后十年，市场已被深度控盘。跟随国家队的配置方向（核心资产、低波红利）比追逐游资票更安全
- **来源**：课程原文 "如果你现货里面配的是国家队的票，就低波红利那些票，那你可能就能够去对冲掉意外来的这种市场结构性的这种风险"

### 规则 21：挣大钱就是熬
- **内容**：底部区域不破前低，就要拿住不动。周期越短越不稳定
- **来源**：课程原文 "挣大钱就是熬"

## 七、认知规则

### 规则 22：交易是持久战
- **内容**：不要指望一两次交易就成了。千锤百炼之后才能知行合一
- **来源**：课程原文 "当你明白了自己的性格跟明白自己的交易策略之后，你真正能产出的时候一定是千锤百炼之后"

### 规则 23：不要碰港股和科创板（新手）
- **内容**：港股单日能跌98%，创业板/科创板单日能跌20-30%，新手不要碰
- **来源**：课程原文 "港股单天能给你跌没了，单天能能跌98%，创业板和科创板那边当天能够跌30%"
- **目的**：保护新手不被极端波动消灭本金

### 规则 24：盈亏同源
- **内容**：能单抡一个票赚大钱的人，未来一定也能单抡一个票亏大钱
- **来源**：课程原文 "你能单轮一次单轮一个票赚到大钱，你以后就百分百能单抡一个票套一个大个的"

### 规则 25：经历痛苦是常态
- **内容**：在N型的低谷进场，连续经历痛苦后淬炼出的心态，才是真正属于自己的
- **来源**：课程原文 "如果你是在N型的最低点的位置进来……你连续经历这种痛苦，那么你会发现你心态已经练就出来之后获得的那种结果，你会觉得那他妈是我应得的"
