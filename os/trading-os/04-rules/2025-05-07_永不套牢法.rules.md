# 交易规则：2025-05-07 永不套牢法

> **课程时间**：2025-05-07
> **课程主题**：永不套牢法——少妇战法实战精讲
> **核心概念**：只输一根K线、BBI两根阳线减仓、顺大势逆小势、大爷图识别

---

## 一、核心纪律

### 规则 1：只输一根K线
- **内容**：买入后以进场K线最低价下方1%或2-3个价位设止损，收盘跌破就走
- **执行**：不考虑区间、不考虑前低、不纠结，破了就是破了
- **来源**：课程原文 "只输这一根K线，记住就输一根K线"
- **备注**：这是永不套牢法的基石，"Z家军永不被套"

### 规则 2：BBI两根阳线减仓一半（卤煮/放飞）
- **内容**：当股价向上脱离BBI线，出现连续两根阳线（中阳线以上），必须减仓一半
- **目的**：降低持仓成本，让剩余仓位拥有绝对成本优势
- **执行后**：半仓盯BBI线持有，不再担心回调
- **来源**：课程原文 "BBI线向上两根阳线一定要减仓一半，把利润给我留下来，向少妇同志学习"

### 规则 3：积小胜为大胜
- **内容**：不追求暴利，每次赚5%-10%，亏损控制在1%-5%，通过复利累积
- **来源**：课程原文 "积小胜为大胜，一路小屁走向胜利"
- **备注**：100万以下资金，一年稳定赚50%完全可行

### 规则 4：永不炫技
- **内容**：下跌趋势的横盘（大爷图）不要碰，宁可踏空不要做
- **来源**：课程原文 "不要炫技，不要炫技，下跌趋势横盘不要碰"
- **备注**：连续两次在大爷图上做错后，第三次即使对了也不值得

## 二、图形判断规则

### 规则 5：上涨趋势横盘可以做
- **内容**：前面有一波明确上涨 → 横盘回调 → 低点抬高 → 下沿B1可以入场
- **来源**：课程原文 "上涨趋势里边的震荡可以做"

### 规则 6：下跌趋势横盘不能碰
- **内容**：连续下跌后横盘、重心向下、BBI向下 → 不参与
- **风险**：横盘末期容易向下破位加速，可能出现一字跌停
- **来源**：课程原文 "下跌趋势里边的震荡不能碰——大爷图"

### 规则 7：大爷图识别标准
- **内容**：连续下跌 + 横盘 + 重心向下 + BBI向下 = 大爷图
- **执行**：直接跳过，寻找下一个标的
- **来源**：课程原文 "大野图记住，连续下跌之后的横盘，而且重心向下，BBI也向下的"

### 规则 8：大爷踏空图识别标准
- **内容**：上涨趋势 + 横盘回调 + 低点抬高 = 大爷踏空图
- **执行**：可以等B1入场（顺大势逆小势）
- **来源**：课程原文 "上涨趋势的这转震荡区间之后的下沿"

## 三、入场规则

### 规则 9：顺大势逆小势入场
- **内容**：在上涨大趋势中，等回调到B1位置时入场；下跌趋势不做多
- **执行**：
  - 判断大趋势方向（BBI向上 = 上涨趋势）
  - 等小趋势回调到B1（KDJ大负值）
  - 设好止损入场
- **来源**：课程原文 "顺大势逆小势"

### 规则 10：B1入场标准
- **内容**：配合KDJ大负值（J值达到大的负值，即"KDJ大幅值"）+ 反转K线确认
- **止损**：进场K线最低价下方1%
- **来源**：课程原文 "勾到大负值了……买完之后设止损，只输一根"

### 规则 11：盈亏比至少3:1才值得做
- **内容**：入场前先算向上空间和向下空间，比例不到3:1不碰
- **来源**：课程原文 "你里外里盈亏比一算，一盈亏比是一一的话，你就没必要干"

### 规则 12：中间位置不碰
- **内容**：横盘区间的中间位置可上可下，没有盈亏比，不参与
- **来源**：课程原文 "中间可上可下，就不要碰"

### 规则 13：第一个跌停不要进
- **内容**：第一个放量跌停即使勾到大负值也不要进，至少等第二天确认
- **来源**：课程原文 "第一个跌停记住……打到勾大幅值也不要进"

## 四、持仓与减仓规则

### 规则 14：BBI线上持股，跌破2日离场
- **内容**：减仓后的半仓以BBI线为持股依据，跌破BBI线连续2日就离场
- **来源**：课程原文 "白线跌破第一天，跌破第二天走了"

### 规则 15：连续大跌后的横盘要警惕加速
- **内容**：连续下跌后勾到大负值的阶段，容易向下加速破位，不要去抄底
- **来源**：课程原文 "任何到最后加速的时候，一定是有先兆的……这种图就不要碰"

### 规则 16：赚钱的票要让它红盘离场
- **内容**：赚钱的单子不能做成亏钱，严格执行BBI两根阳线减仓一半
- **来源**：课程原文 "赚钱的票一定要让它红盘离场"

## 五、仓位规则

### 规则 17：100万以下分4-5个仓位
- **内容**：每次交易只用1/4到1/5的仓位，单票止损控制在1%-5%
- **来源**：课程原文 "10万块钱分成五个仓，每个仓2万块钱"

### 规则 18：不要盯盘中，只看收盘
- **内容**：少妇战法不需要盯盘，收盘前十分钟看一眼即可
- **来源**：课程原文 "不要盯盘，没有必要，一切的一切都是在收盘价上"

### 规则 19：没有机会不要强行交易
- **内容**：市场没有明确B1机会时，空仓等待
- **来源**：课程原文 "没有B一，没有B一，那这波就不要干了"

## 六、认知规则

### 规则 20：踏空比亏钱更难受
- **内容**：永远要有一定的底仓，空仓的踏空感会导致乱操作
- **来源**：课程原文 "踏空比你亏钱要难受多得多"

### 规则 21：下跌趋势不炫技
- **内容**：下跌趋势即使有反弹机会也不做，胜率太低
- **来源**：课程原文 "大爷图你不要炫技，不要劝"

### 规则 22：最后一笔交易一定会失败
- **内容**：任何连续成功的模式，最后一定有一次止损出局，要接受
- **来源**：课程原文 "你最后一定会输一次，你这次进了，你这个就输掉了。输掉赶紧走，只输一根K线"

### 规则 23：少妇战法只适合小资金
- **内容**：100万以下用少妇战法积小胜为大胜；500万以上应做长线趋势
- **来源**：课程原文 "100万以下你就这么轮，500万以下就按盘子大的票轮"

### 规则 24：优先做上涨趋势的票
- **内容**：选择上涨趋势的票做B1，不做下跌趋势的反弹
- **来源**：课程原文 "为什么少妇能赚钱？只做上涨趋势的票"
