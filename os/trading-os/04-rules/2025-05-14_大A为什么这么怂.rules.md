> **核心概念**：A股底层认知、少妇战法六步、止损三级别、洗盘识别、合并同类项

## 一、A股底层认知规则

### 规则 1：A股六点底层认知
- **内容**：A股长期疲软有六个根本原因，需建立底层认知
  1. **长期充分博弈**：各方参与者极致内卷，谁都不让谁
  2. **交易机制不对等**：机构T+0、散户T+1，各种门槛形成套利空间
  3. **老麒麟会根深蒂固**：每个老票背后都有主，5-10年做一波
  4. **洋务派主导大局**：留学背景基金经理看美股脸色，不敢独立表态
  5. **政策驱动机制**：国家队和制度安排引导市场方向
  6. **独特盈利模式**：不需要长期牛市，只需波动中获利
- **来源**：课程原文 "大A为什么这么怂，有六个原因"

### 规则 2：不要对A股抱不切实际幻想
- **内容**：10年内不会再有924，不要期待单边牛市
- **来源**：课程原文 "10年内不会再有924了"

### 规则 3：A股的盈利模式 = 波动
- **内容**：A股不需要从3000到6000，只需要从3000到2600再到3300
- **策略对应**：波动市场最适合低买高卖，对应少妇战法
- **来源**：课程原文 "大A的盈利模式不需要单边上涨，它只需要波动"

## 二、少妇战法六步执行规则

### 规则 4：第一步——择时
- **内容**：判断当前市场阶段（上涨/下跌/横盘），决定是否参与
- **说明**：短期择时较难，可借助周日宏观直播判断
- **来源**：课程原文 "择时这件事我还没有教过……短期择时你就听我直播就完了"

### 规则 5：第二步——选股
- **内容**：基本面看政策 → 政策看行业 → 行业看龙头 → 盘面语言确认
- **原则**：不离开主线，不碰地产等下跌趋势板块
- **来源**：课程原文 "选股包括是基本面先看政策，政策上看行业，行业里面看龙头"

### 规则 6：第三步——等B1
- **内容**：不追高，等KDJ大负值（即"KDJ大幅值"）的B1入场
- **来源**：课程原文 "等B1，一定要等B1"

### 规则 7：第四步——设止损
- **内容**：进场后必须设止损，按碗的大小分三个级别
- **止损三级别**：
  - **级别一（敏感型）**：进场K线最低价下方3-5个价位，最保守
  - **级别二（趋势型）**：本阶段最低点或上个阶段最低点
  - **级别三（盈亏比型）**：算好盈亏比（如向下10%向上50%），不看具体点位
- **原则**：一切以收盘价为准
- **来源**：课程原文 "止损里边分三个阶段"

### 规则 8：第五步——止盈放飞
- **内容**：BBI线上出现两根中阳线，减仓一半
- **中阳线标准（因票而异）**：
  - 低波动票（中石油）：3%即为中阳线
  - 券商：4%-6%为中阳线
  - 台子：3.5%即为中阳线
  - 青花：4%-6%为中阳线
  - 创业板（20cm）：7%才叫中阳线
- **来源**：课程原文 "不同的票波动率不一样，中阳线标准也不同"

### 规则 9：第六步——收队
- **内容**：跌破BBI线两根K线，全仓离场
- **来源**：课程原文 "跌破BBI线两根K线收队"

## 三、入场规则

### 规则 10：B1第二天开盘买
- **内容**：B1信号出现后，一般第二天开盘买入，不用当天尾盘追
- **原因**：连续下跌后第二天通常顺势平开或低开
- **例外**：特噗战法有时需要收盘买（第二天容易高开）
- **来源**：课程原文 "要是B1的话，我一般是第二天开盘买"

### 规则 11：洗盘识别——价格不跌指标跌
- **内容**：当股价没怎么跌但KDJ快速下到大负值时，是洗盘不是出货
- **执行**：洗盘位置是B1买入良机，不要恐慌
- **来源**：课程原文 "股价没怎么下跌，但是指标快速下跌了，这叫洗盘"

### 规则 12：合并同类项——回调加仓
- **内容**：已有大幅浮盈，回调时KDJ再次大负值，可加仓
- **执行**：新旧仓位统一设止损在加仓K线最低点
- **条件**：有大幅浮盈 + 回调到B1 + KDJ大负值
- **来源**：课程原文 "合并同类项……有浮盈，大幅浮盈……勾是大负值了，把大负值的这一根K线的最低点往下设止损"

### 规则 13：上涨趋势转震荡后，最后一次必止损
- **内容**：每个上涨趋势变成横盘后，最后一次B1操作一定会止损出局
- **心态**：接受这是交易的一部分
- **来源**：课程原文 "每一个上涨趋势变成横盘趋势之后的最后一次，你总是会要止损的"

## 四、持仓规则

### 规则 14：行情末期不再做第二次
- **内容**：完整波段走完后，转震荡了就不再参与，等下一个新机会
- **来源**：课程原文 "一般来讲一波能明显走完的情况下，转势做一次漏了就不管了"

### 规则 15：趋势未终结可加仓
- **内容**：上涨趋势未破坏的横盘回调，连续B1可做（双B1），盈亏比高
- **来源**：课程中的多个案例演示

### 规则 16：少妇战法各阶段胜率
- **内容**：
  - 上涨趋势 + 选股择时正确：七成以上胜率
  - 赶上行情：九成以上胜率
  - 单边下跌趋势：胜率低但不会追到反弹最高点
- **来源**：课程原文 "择时选股都把握问题不大的情况下，七成以上胜率"

## 五、认知规则

### 规则 17：永远保持如履薄冰
- **内容**：赚钱时不要觉得成了，市场远没走好
- **来源**：课程原文 "永远要如履薄冰，永远要战战兢兢"

### 规则 18：追高是长期必亏的行为
- **内容**：追高10次有8次熄火，即使某次追对了也是错的
- **来源**：课程原文 "你只要长期追，你追十次八次都熄火"

### 规则 19：遵守纪律比单次盈亏更重要
- **内容**：今天追了即使浮盈，这笔交易也是错的，因为违背了纪律
- **来源**：课程原文 "就算明天大涨了让你今天追的这个出现浮盈了，你这把交易也是错的"

### 规则 20：不离开主线
- **内容**：每个阶段市场有明确的主线，偏离主线容易踏空或套牢
- **来源**：课程原文 "永远不要离主线太远"

### 规则 21：少妇战法口诀
- **内容**：把六步口诀背下来形成肌肉记忆
  - B1买 → 设止损 → 破了平 → 涨起来BBI线上两根中阳线放飞一半 → 看BBI线 → 跌破两根全离场
- **来源**：课程原文 "就像背口诀一样，就每天放到自己的脑中，一定要反复的"

### 规则 22：回测要分场景
- **内容**：策略回测要分上涨/下跌/横盘三段分别测，不能混在一起
- **来源**：课程原文 "你要把它至少分成三部分，上涨、下跌、横盘"
