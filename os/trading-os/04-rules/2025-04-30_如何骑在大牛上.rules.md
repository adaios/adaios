# 交易规则：2025-04-30 如何骑在大牛上

> **课程时间**：2025-04-30
> **课程主题**：如何骑在大牛上——K线、均线与核心指标入门
> **核心概念**：机构思维、K线基础、BBI趋势判断、MACD顶底背离、B1/B2入场

---

## 一、基础认知规则

### 规则 1：机构思维 vs 散户思维
- **内容**：摆脱散户思维（一调整就崩了、买了必须涨、不能跌），建立机构思维
- **机构思维特征**：
  - 区分正常回撤和崩盘
  - 知道每个票的配置角色
  - 严格止损
  - 不因情绪波动做决策
- **来源**：课程原文 "您跟我这儿听的是一个机构思维"

### 规则 2：7%以上才叫崩
- **内容**：跌幅7个点以上才叫崩，跌1.6%叫正常波动，不是崩
- **来源**：课程原文 "就在我的概念里边，就是跌幅七个点以上那叫崩"
- **备注**：连续两个跌停才叫系统性风险

### 规则 3：不要满仓单调一只票
- **内容**：不要重仓单抡一个票，必须配置
- **来源**：课程原文 "咱家聊了一年半了，一直在说不要加杠杆做票，不要满仓单，一定要配置"

### 规则 4：回撤 vs 反弹
- **内容**：
  - 上涨趋势中的阴线叫**回撤**，是正常现象
  - 下跌趋势中的阳线叫**反弹**，到压力位必须出
- **来源**：课程原文 "下跌趋势中的阳线叫反弹，到压力位必须出"

### 规则 5：用输得起的钱交易
- **内容**：计算每个持仓两个跌停对净值的影响，如果影响心态说明仓位过大
- **来源**：课程原文 "你配任何一票自己去算两个跌停对你净值影响能不能影响你整个人的心态"

## 二、K线规则

### 规则 6：K线基本原则
- **内容**：
  1. 先看K线范围（最高-最低），再看实体大小
  2. 收盘再看K线，不到收盘不知道多空胜负
- **来源**：课程对K线的系统讲解

### 规则 7：K线多空含义
- **内容**：
  - 阳线：收盘价>开盘价，多方胜
  - 阴线：开盘价>收盘价，空方胜
  - 最高价：多方最大战果
  - 最低价：空方最大优势
  - 收盘价：当日最终胜负
- **来源**：课程原文

### 规则 8：健康上涨结构
- **内容**：低点抬高 + 高点抬高（N型上涨）为健康上涨
- **确认方式**：高点突破 + 低点不破
- **来源**：课程原文 "低点抬高，高点抬高，上涨趋势"

### 规则 9：重要K线形态判断
- **内容**：
  - 长下影阳线：多方完爆空方，看涨
  - 长上影阳线：多头试盘遇阻，守实体低点可看涨
  - 长下影阴线：空头试盘遇抵抗，不破实体高点继续看跌
  - 长上影阴线：空头完爆多方，看空
  - 十字星（小阳/小阴十字星）：多空势均力敌，趋势末端易反转
  - 标准K线（无影线/适中实体）：趋势延续
- **来源**：课程对K线形态的系统讲解

## 三、趋势判断规则

### 规则 10：BBI判断趋势方向
- **内容**：
  - BBI向上 → 持股
  - BBI向下 → 离场
  - BBI走平 → 减少操作，等待方向
- **注意**：BBI看趋势不看金叉死叉
- **参数**：默认3-6-24即可，无需修改
- **来源**：课程原文 "BBI是判断趋势的"

### 规则 11：中线赚大钱只看BBI
- **内容**：中线策略忽略中间K线波动，只看BBI方向
- **来源**：课程原文 "中线赚大钱只看BBI，忽略中间K线"

### 规则 12：右侧交易原则
- **内容**：做右侧交易一定要等趋势走出来再入场
- **来源**：课程原文 "做右侧一定要最好起来……左侧可以玩或者短线，但右侧一定要走起来"

## 四、入场规则

### 规则 13：B1入场（N型下跌转折）
- **内容**：在N型下跌结构的转折点（B1）进场
- **结构特征**：跌 → 反弹 → 再跌 → B1转折
- **依据**：配合KDJ大负值（J值达到大的负值，即"KDJ大幅值"）+ MACD底背离
- **来源**：课程原文 "N型这块出来了……在N型的这个拐点位置"

### 规则 14：B2入场（趋势确认后）
- **内容**：B1之后趋势确认后的第二个买入点
- **特征**：
  - 比B1确定性更强（趋势已经确认）
  - 止损幅度大（可达7%-8%）
- **建议**：未熟练掌握B1止损前，不建议做B2
- **来源**：课程原文 "B2比B1确定性更强……止损幅度大"

### 规则 15：MACD底背离 + BBI向上 + KDJ大负值 = 做错也要做
- **内容**：三个指标同时满足条件时，确定性强，放手执行
- **来源**：课程对多个指标配合使用的讲解

### 规则 16：深水炸弹入场
- **内容**：连续大幅下跌后ADX拐头向上，博超跌反弹
- **执行**：必须严格止损，3天不涨必走
- **适用**：日线级别，用于"屁滚尿流"的连续下跌之后
- **来源**：课程原文 "深水炸弹……你一定要记住屁滚尿流"

## 五、离场/止损规则

### 规则 17：MACD顶背离减仓
- **内容**：股价创新高但MACD红柱明显缩短或绿柱出现，必须减仓
- **两种识别方式**：
  1. 黄线位置一次比一次低
  2. 同一波上涨中，价格更高但红柱高度反而更小
- **来源**：课程原文 "MACD顶背离几乎无失败案例"

### 规则 18：只输一根K线
- **内容**：止损设在进场K线最低点下方几个价位
- **原理**：收盘跌破进场K线最低价 = 判断错误，必须离场
- **来源**：课程原文 "入完以后当根设止损……往下定22 20 22.4块，定在这儿止损"

### 规则 19：BBI拐头向下减仓
- **内容**：BBI拐头向下 + 连续2日收盘收不回 = 减仓或离场
- **来源**：课程的BBI应用

### 规则 20：顶部形态宁可信其有
- **内容**：出现顶部形态信号时，宁可信其有不可信其无
- **来源**：课程原文

## 六、持仓规则

### 规则 21：中长线分批买入
- **内容**：长线票可以分几批慢慢买；短线票一般就买这一天，两笔单三笔单敲进去设好止损
- **来源**：课程原文 "长线的票分几批，短线的票一般就买这一天"

### 规则 22：BBI走平阶段 = 减少操作
- **内容**：BBI走平时交易价值低，减少操作，等待方向
- **来源**：课程原文 "BBI走平就是非常无聊的阶段"

### 规则 23：不要碰超小盘/ST
- **内容**：未来1-2个月有45家退市、70多家ST，别碰超小盘，别赌ST重组
- **来源**：课程原文 "现在千万别别碰这些票……没人炒ST摘帽没人炒重组"
- **备注**：这波退市是来真的，不要抱幻想

## 七、认知规则

### 规则 24：机构思维 = 闷声发大财
- **内容**：不听古巴/舆论影响，跟机构思维走
- **来源**：课程原文 "不用被他们影响，踏踏实实的咱们自己做咱自己的"

### 规则 25：散户唯一赢面 = 中线以上策略
- **内容**：散户唯一能跟机构周旋的方式是中线以上的策略，短线面对的是量化/麒麟会/经验更丰富的交易者
- **来源**：课程原文 "对于绝大部分的散户来讲，你唯一能够跟机构去稍微的周旋一下，就是至少中线起了"
