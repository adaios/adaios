## 第零层：底层认知

> A股为什么是今天这个样子，谁在参与，钱往哪去。

### 0.1 A股市场底层逻辑
- A股定位：融资市 → 投资市
- 国家意志决定论（形的手 > 无形的手）
- A股三大本质：政策市、资金市、技术市（04-04）
- 三辞三让（国际博弈策略，04-04）
- 顺周期启动顺序（银行→券商→科技→保险→消费）
- 信息安全鉴别（消息不决定盘面，04-06）
- 财政与股权绑定（龙队的钱不能输）
- 财富再分配两个途径（新故事印新钱 / 通胀发钱+税收回收）

### 0.2 市场参与者认知
- 机构 vs 游资 vs 量化
- 跟庄三条件（让跟/蹭跟/帮跟）
- 建仓洗盘拉升出货四步模型
- 机构做逻辑 vs 散户做情绪（04-20）
- 产业资本视角：股东思维，老板在给你打工（04-20/04-26）
- 投资者成长路径：短线→波段→压行业做配置（04-26）
- 公募行为框架
- 麒麟会洗盘博弈

### 0.3 新故事方向（随行情更新）
- 主题共识形成路径（标签→标的→参与人→合力→容纳大资金→机构跟进→主线）
- 池子方向（主线→支线→答应的结构，04-12）
- 产业链投资逻辑：初期（纯故事/炒想象力）→成长（确定性强）→成熟（抗通胀/高分红）

## 第一层：择时（什么时候做）

> ⭐ 全部模块中最重要的一层。择时不对，一切白费。

### 1.1 择时三要素
- 基本面择时 → 宏观周期
- 资金面择时 → 增量资金
- 技术面择时 → K线形态、MACD
- 展开为天时·地利·人和框架

### 1.2 活跃市值（0AMV）体系
- 0AMV 指标来源与用法
- 增量资金入场分级：
  - 小波段：4%+阳线 = 主题级别
  - 大波段：5%-8%阳线 = 主线级别
  - 十年一遇：7%-20%阳线 = 全市场普涨
- **2.3%离场法则**（-2.3 绿柱 = 减仓）
- 增量资金 vs 存量资金切换
- **活跃市值本质：筹码与钱的关系**（04-06）
- **筹码沉淀**：924底仓56000→17万（04-06）
- **浮筹清洗**：新行情前必须把参与者洗走（04-06）
- **强势股补跌是新行情前奏**（04-06）
- **早涨早跌早反弹**（高切低，04-06）
- **空头区间只卖不买**（04-04/04-08）
- **金叉空守一波**（活跃市值金叉但已转空，04-04）

### 1.3 MACD三种用法（04-22）
- **零轴多空**：白线在上=多头，在下=空头
- **顶背离与底背离**：背离之后还能再背离
  - 底背离看周线更可靠，顶背离看日线
  - 机构左侧入场在底背离，散户右侧确认后买入
- **金叉空与死叉多（空中加油）**：源于1970年代期货技术
- **本质逻辑**：顺周期=建仓波（真买）→洗盘→筹码锁定→不花钱也能拉

### 1.4 择时原则
- 宽松周期+经济拐点：不能空仓
- 风险>收益：空仓也是策略
- 策略回测须分三区间（上涨/下跌/横盘）
- 10年内不会再有924
- **时间换空间**（04-06）：用阴跌磨底消化浮筹

> **对应课程：** 0423、0514、0702、0827、0903、1027、0404、0406、0422

## 第二层：选股与配置（做什么票）

### 2.1 选股逻辑链
```
基本面看政策 → 政策看行业 → 行业看龙头 → 盘面语言确认
```

### 2.2 配置模式
- **池子（当前）**：主线（AI算力存储半导体芯片）→ 支线（光伏储能新能源）→ 答应（创新药）
- **曼城阵容（历史）**：4231 固定比例
- **两条路线**：主仓（左仓拿住不动）+ 副仓（右仓少妇战法做增强）

### 2.3 核心原则
- 不离开主线
- 不碰下跌趋势板块（大爷图/大野图）
- 主题 vs 主线投资区分
- 填坑出坑为核心战法
- 坑不能做三条件
- 机构资金定价权排序（国家队→社保→保险→公募→券商→私募→麒麟会→游资→散户→外资）
- **全力追一个方向要小心见顶**（04-12）
- **选择比努力重要，站在产业角度**（04-26）

### 2.4 出货验证（04-08/04-12）
- 筹码理论：低位→高位转移 = 出货
- 股东户数验证：底部户数少→高位大增=倒给散户
- S1优先级高于筹码理论，先保命

> **对应课程：** 0423、0531、0625、0626、0415、0806、0813、0922、0408、0412、0426

## 第三层：入场方式（怎么买）

### 3.1 核心买点体系

| 买点 | 条件 | 说明 |
|:----|:-----|:-----|
| **B1** | N型转折 + KDJ大幅值（J<13）+ 缩量确认 | 核心入场方式，不是抄底 |
| **B2** | B1之后的阳线确认 | 更好，止损更大 |
| **B3** | B1→B2完整波段后；B2之后的大阳线或缩量阳小十字星（代表一致） | 只看白线（牵牛绳），99人不做 |
| **SB1** | 缩量横住→放量下跌→再缩量横住→勾大幅值 | 主升前最后一洗 |

### 3.2 特殊入场方式
- 暴力特噗（突破入场）
- 深水炸弹（超跌反弹，ADX拐头）
- 单针下20（公式指标）
- 补票战法（B1错过后的补票方案）

### 3.3 买入三问
1. 买入预期是什么？
2. 盈亏比划不划算？（短线至少 3:1）
3. 哪里设止损？

### 3.4 B1/B2风控
- **B2不能有大苗子**（B2是攻击型阳线，04-29）
- **建仓波大苗子正常**（建仓波是买不是攻击，04-29）
- 没看懂就不干，天天空仓不亏钱（04-29）

> **对应课程：** 0423、0430、0507、0514、0531、0601、0716、0415、0806、0819、1006、0429

## 第五层：应对（买了以后怎么办）

> ⭐ 按优先级从最紧急到最终，逐级不跳。

```
没涨 → 盈转亏 → 止损 → 放飞 → S1 → 滴滴 → 破白线 → 破黄线 → 死叉
```

### 5.1 没涨（最重要）
- 买入后没脱离成本区
- 处理：拍掉换下一批
- "市场没有证明你是对的时候，你就错了"

### 5.2 盈转亏
- 盘中浮盈变浮亏 → 不等收盘，直接拍掉

### 5.3 止损体系
- **只输一根K线**（核心理念）
- 止损三级别（敏感型/趋势型/盈亏比型）
- 止损三原则
- 白线（知行趋势线）拐头止损、N型止损、关键K止损
- 每日压力测试（两个跌停不影响睡觉）

### 5.4 止盈放飞
- 白线（知行趋势线）上两根中阳线 → 减仓一半
- 第四块砖放飞铁律
- **金叉空位置的滴滴最有参考价值**（04-22）
- 浮盈多时敢于放飞（03-25）

### 5.5 见顶出货信号
- S1（五种出货方式）
- 顶部大风车
- MACD顶背离（只在放巨量后有判断价值）
- 滴滴（阶梯量出货）

### 5.6 破线应对
- **破白线**：短期趋势大概率走坏
- **破黄线**：大哥不在
- **死叉**：需要时间修复

### 5.7 收队规则
- 跌破白线（知行趋势线）两根 → 全仓离场
- 顶部形态宁可信其有
- 单顶不是顶

> **对应课程：** 0507、0514、0709、0712、0716、0415、0819、0824、0422
