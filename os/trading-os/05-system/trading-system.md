## 总纲：少妇战法六步

```
①择时 → ②选股 → ③等B1 → ④设止损 → ⑤止盈放飞 → ⑥收队
```

**执行口诀：**
```
B1买 → 设止损 → 破了平 → 涨起来白线（知行趋势线）上两根中阳线放飞一半 → 看白线（知行趋势线） → 跌破两根全离场
```

**两条路线（12-17）：**
- **底线**：少妇战法六步，靠交易赚波段的钱
- **上限**：主仓拿住不动（赚大钱）+ 副仓少妇战法（做增强），两条腿走路（12-31）

## 第一步：择时（什么时候做）

### 1.1 底层认知（A股为什么这么"怂"）

**A股底层六点：**
1. 长期充分博弈 → 各方参与者极致内卷
2. 交易机制不对等 → 机构T+0 vs 散户T+1
3. 老麒麟会根深蒂固 → 每个老票背后都有主
4. 洋务派主导大局 → 留学背景基金经理看美股脸色
5. 政策驱动机制 → 国家队和制度安排引导方向
6. 独特盈利模式 → 不需要单边牛市，只需波动中获利

**扩展认知（后课补充）：**
- **财政与股权绑定**（12-31）：以前财政与地产绑定，现在转向与股权绑定。"上有顶下有底"不是空话，龙队的钱不能输
- **A股本质是货币现象**（11-23）：双宽政策+存款未搬家+美国不缩表=牛市基石不变。存款搬家才是大盘见顶信号
- **大A不会踏空**（01-05）：长期参与者不必因一两天上涨而焦虑
- **顺周期启动顺序**：银行→券商→科技→保险→消费

### 1.2 择时三要素

1. **基本面择时** → 判断大方向（宽松/紧缩，经济周期位置）
2. **资金面择时** → 看增量资金（活跃市值为最核心指标）
3. **技术面择时** → 找具体买点（K线形态、MACD、成交量）

### 1.3 活跃市值体系（核心择时工具）

**信号规则：**
- **4%启动信号**：单日≥4%或连续两日累计≥4%=新波段开始
- **-2.3%离场法则**：单日跌幅≥2.3%→大概率波段见顶
- **-2.3%后策略**：转为"只卖不买"，个股没走完的继续等S1（02-04）
- **重心下移**：连续数日重心仍下移=不是猛干的时候（11-17）
- **跳空低开**：活跃市值跳空低开放量下杀≈阶段性底部临近（11-23）
- **百点长阴时机构出不去货**，散户不必恐慌（02-04）
- **活跃市值本质是筹码与钱的关系**（04-06）：不是简单的资金流入流出，而是全市场流通筹码的货币化价值。每笔交易都是"筹码变钱"或"钱变筹码"
- **筹码沉淀**（04-06）：924底仓56000→17万，大量长线锁仓是龙队控指数的底层基础
- **浮筹清洗**（04-06）：新行情启动前必须把之前参与者洗走——成交量极致缩量→锁仓拉升
- **强势股补跌是新行情前奏**（04-06）：最后一波下跌必然是强势股补跌，不是千股跌停。强势票加速补跌=底部临近
- **早涨早跌早反弹**（04-06）：先涨的先跌，先跌的先弹。提前调整的板块在空头区间末期可能逆势反弹。

**空头区间 vs 多头区间（03-30）：**
| 区间 | 特征 | 操作 |
|:----|:-----|:-----|
| 多头区间 | 活跃市值红柱上升 | 容错率高，B1随便做，大胆做 |
| 空头区间 | 活跃市值绿柱下降 | 收紧策略，不出手等信号 |
| 切换确认 | 零轴之上二次金叉（03-01） | 转多头思维 |

**2026节奏预判（03-01）：**
```
一月份打窝 → 二月份调整 → 三月份二次金叉主升 → 四月份收队 → 六月份再说
```

**其他择时信号：**
- 12月中旬铁蝴蝶开始入场布局下一年主线（12-01）
- MACD探底法：零轴上方下穿到零轴下方越往下越好
- 2026目标区间：4700~4800才是真正需要恐慌的区域（01-05）

### 1.4 MACD三种用法（04-22）

**用法一：零轴多空**
白线在零轴以上=多头区间，零轴以下=空头区间。先看白线再看红柱。线是市场合力的趋势，柱可以画出来（对倒）。

**用法二：顶背离与底背离**
价格创新高但白线没创新高=顶背离。价格创新低但白线没创新低=底背离。背离之后还能再背离。
- 底背离看周线更可靠，顶背离看日线
- 机构左侧入场在底背离，散户右侧确认后买入

**用法三：金叉空与死叉多**
金叉空=白线马上要金叉但拐头向下（下跌延续）。死叉多=白线马上要死叉但拐头向上（上涨延续）=空中加油。

**本质逻辑：** 顺周期=建仓波（真买）→洗盘→筹码锁定→后面不花钱也能拉。完整周期：建仓→洗盘→初升→洗盘→主升→出货→MACD变绿→空头区间。

### 1.5 择时原则

- 宽松周期+经济拐点：**不能空仓**（至少半仓）
- 判断风险>收益：**空仓也是策略**——零仓位也是交易决策（02-02）
- A股盈利模式=波动，最适合少妇战法
- 10年内不会再有924，不抱不切实际幻想
- 银行拉新高=龙队定价慢牛方向；银行不新高=危险信号（11-12）
- 券商≠行情发动机，券商是龙队控节奏的工具
- **时间换空间**（04-06）：主力不选择A杀，用阴跌磨底消化浮筹。散户体感差但指数不崩。对应策略——跟着磨，至少还能做一波两个月的行情

### 1.6 策略回测原则

策略必须分**上涨/下跌/横盘**三个区间分开测试，没有策略能在三种行情中都赚钱。

## 第二步：选股（做什么票）

### 2.1 选股逻辑链

```
基本面看政策 → 政策看行业 → 行业看龙头 → 盘面语言确认
```

### 2.2 核心原则

- **不离开主线**：每个阶段市场有明确主线，偏离容易踏空
- **不碰下跌趋势板块**（地产等大野图）（03-11重申：地产全线空头，保利走好前不看）
- **选不出来=进步**（12-03）：无聊行情选不出B1说明审美提高了，宁缺毋滥
- **板块龙头验证法**（03-11）：板块要有行情，大盘龙头必须先翻番

### 2.3 两条路线

**路线A：主仓+副仓（12-31最终版）**
- 主仓（左仓）：确定性强的票拿去不动，睡得好觉
- 副仓（右仓）：少妇战法做交易增强，赚波段钱
- **主仓副仓分离**：不能把副仓拿成主仓，也不能把主仓在低位拍掉（12-17）

**路线B：曼城阵容（大资金/中线）**
- 4231阵型：40%低波红利防守+30%前锋进攻+20%中场过渡+10%机动备用
- 每个票有明确角色（攻/守/对冲）
- 弱势票拍掉标准：跌破黄线很久的票清理出阵容（12-01）

**路线C：基石+前锋（12-08升级版）**
- 60%基石（防守/渡调整）+ 40%前锋（增强/进攻）
- 从防守转进攻、从进攻转防守要有明确的切换逻辑

### 2.4 板块方向（按2026框架）

| 层级 | 方向 | 配置角色 |
|:----|:-----|:---------|
| 趋势中枢 | 半导体（六大攻关） | 长线底仓 |
| 落地方向 | AIGC应用（十四五→十五五落地） | 进攻前锋 |
| 核心防守 | 白酒/煤炭 | 基石防守 |
| 辅助方向 | 创新药、储能、机器人 | 波段增强 |

- **AIGC三段论**（12-01）：纯血（港股阿里/队长）→混血（A股ETF）→特色标的
- **白酒+AI双配置**（02-11）：防守端白酒+进攻端AIGC
- **板块分散原则**（01-07）：与其给一个板块十次机会，不如给十个板块一次机会

### 2.5 ETF选行业

当无法确定个股时，用ETF吃平均收益（12-01）：
- 选偏软件不偏硬件→偏应用不偏游戏→偏AIGC不偏算力
- ETF入场也用B1体系（周线B1建仓法：小仓位打底→往下漏再加→异动回踩推满）

## 第三步：入场（怎么买）

### 3.1 B1（核心入场方式）

```
条件：连续下跌→N型转折→KDJ大负值→缩量确认
```
- B1不是抄底，是确认转折后才介入
- 配合MACD底背离+白线（知行趋势线）向上，确定性更强→做错也要做
- 一般第二天开盘买入

**B1递进确认链（02-24）：** B1→再B1→SB1，不要一个B1就满仓，等后续确认再加仓。
**B1要的是爆发力（02-24）：** 脱离成本区要快，缩量B1可靠性低于放量B1。
**B1三天原则（数据回测）：** 3天内起来就起来了，第4天开始概率衰减。

### 3.2 B2 / B3 / SB1

| 买点 | 盈亏比 | 上涨确定性 | 适合 |
|:----|:------|:-----------|:-----|
| B1 | 最高 | 中 | 所有人 |
| B2 | 高 | 高 | 有经验者 |
| B3 | 中 | 最高 | 高手（99人不做）|
| SB1 | 中 | 很高 | 忍出来的买点 |

**B2止损**：第二天K线低点破掉就走（滴滴规则）。
**B3**：B2之后的大阳线，或缩量阳小十字星，代表多空一致（04-29补充定义）。翻倍票第三个B1假破位往往是深蹲起跳的真机会。

### 3.3 特殊入场方式

**底部暴力K线体系（12-24核心）：**
| 类型 | 定义 | 交易意义 |
|:----|:-----|:---------|
| 大暴力·趋势扭转 | 长期下跌后第一根放量大阳 | 只用作观察 |
| 大暴力·趋势改变 | 横盘后改变方向的放量大阳 | 可操作，配黄线站上+关键K确认 |
| 小暴力·趋势延续 | 上涨中回调到第一位置 | **最具操作价值** |

**四条选股原则：** 底部位置+突兀/莫名其妙+又粗又大+让人害怕的时间点。

**其他入场方式：**
- **暴力特噗**：放量大阳线突破前高，特噗后不能出现3/4阴量线
- **深水炸弹**：连续大幅下跌+ADX拐头向上，3天不涨必走
- **单针下20/单针下30**：知行洗盘短线副图指标的白线（散户线）被打到20或30以下的信号。数值越小要求洗得越狠。公式见 `09-scripts/trading-formulas.md`
- **补票战法**：B1错过后的补票方案，突破后入场

### 3.4 洗盘识别

- 股价没怎么跌，但KDJ快速下到大负值=洗盘不是出货→B1买入良机
- **击穿对手盘**（12-31）：主力洗盘不是针对散户，是针对混进来的大资金。消耗耐心+制造恐惧两种手法（01-05）

### 3.5 假突破识别

- 大A市场90%以上的突破是假突破（03-01），量化专门反打突破
- **应对**：别做突破，买B1。在N型结构低点买B1比追突破舒服得多

## 第四步：设止损（怎么防）

### 4.1 只输一根K线（核心理念）

```
止损设在进场K线最低价下方几个价位（或1%），收盘跌破就走
```
永不套牢。

### 4.2 止损三级别

| 级别 | 设置方法 | 适用 |
|:----|:---------|:-----|
| 敏感型 | 进场K线最低价下方3-5个价位 | 新手/小资金 |
| 趋势型 | 本阶段最低点或上个阶段最低点 | 有经验者 |
| 盈亏比型 | 按盈亏比算 | 大资金 |

### 4.3 多层防守体系（后课积累）

```
S1（天量高位）→ 滴滴（阶梯量）→ 破白线（牵牛绳）→ 破黄线（大哥线）→ 死叉
```

- **白线破掉看黄线**（02-11）：白线是短期趋势线，黄线是最后的防守线
- **高位跌破黄线就走**（03-04）：特别是放量跌破黄线，不要扛
- **跌破黄线第二天确认再走**（12-24）：可以不教条，取决于承受力
- **假阴真阳处理**（01-14）：一律当真看，等放量或缩量突破确认
- **缩量突破=控盘**：筹码都在主力手里，后续看涨
- **金叉空位置的滴滴最有参考价值**（04-22）：当MACD金叉空信号出现后的阶梯量（滴滴）最值得重视——协和医院级别的多空确认

### 4.4 其他止损方式

- N型止损、白线（知行趋势线）拐头止损、关键K止损、成本没优势设白线止损（03-16）

## 第五步：止盈放飞（怎么减）

### 5.1 核心规则

```
白线（知行趋势线）上出现两根中阳线 → 减仓一半（卤煮/放飞）
```
剩余半仓以白线（知行趋势线）为持股依据。

### 5.2 涨势延续的减仓节奏

- 第一波减半后继续大涨可再减至1/4
- **涨停板放飞原则**（12-24）：B1入场的涨停板放不放飞都可以，二次封板坚决可以不飞
- **DSZ战法/1234战法**（03-18）：连续四连涨后收绿≈高概率调整，第8根K线考虑减仓
- **4%以上酌情持有或放飞**（02-19）
- **主升中敢于放飞**（03-25）：浮盈多的时候不要该飞不飞

### 5.3 见顶信号

- **顶部大风车**（一根K线）：放天量+长上影+长下影+螺旋桨，走错也要走
- **S1/S2/S3逃顶体系**：加速放量→双头顶背离→放量下沿反弹确认
- **单顶不是顶**：大盘股一天出不完货，有二次冲高机会
- **MACD顶背离**：只在放巨量之后才有判断价值

## 第六步：收队（怎么走）

### 6.1 核心规则

```
跌破白线（知行趋势线）两根K线 → 全仓离场
```
一切以收盘价为准。不和股票谈恋爱。

### 6.2 出场原则

- **顶部形态宁可信其有**
- **单顶不是顶**（大盘股有二次冲高）
- **下跌中的阳线叫反弹**：到压力位必须出，不是反转
- 行情末期不做第二次

### 6.3 春季行情收队（03-30）

收队是减仓到防守仓位，不是全部清掉。空头区间不要干，等下一波信号。

### 6.4 孤注一掷不可怕（03-30）

如果一笔交易输了就伤筋动骨，就不要做。

## 补充A：认知与纪律

### A1 四不原则
```
不追 → 不动 → 不慌 → 不乱摸
```

### A2 核心认知（按重要程度排列）
- **赚大钱 vs 赚小钱**（12-31）：少妇战法赚交易钱，左仓赚认知钱。两条路不能混用
- **杨幂理论**（12-31）：持仓不涨但出现更完美图形时无条件切换
- **习惯打败人性**（12-31）：不需要克服人性，多做几次形成肌肉记忆
- **心态是交易核心**（03-18）：走势为打崩你心态而设计
- **消息面都是噪音**（03-18）：上涨是主力在干活，下跌是主力要走
- **先不死原则**（12-10）：第一原则不是赚钱，是保证不亏大钱
- **交易应对比预测重要**（04-20）：到了位置就应对，不比预测。不基于假设操作
- **强者恒强**：做冠军粉
- **不给第二次机会**：能套你一次就能套你两次
- **只为最完美而停留**（12-24）：短线重仓核心纪律
- **没看懂就不干**（04-29）：天天空仓也不亏钱。确定性能提高胜率
- **高位三波不做**（04-29）：涨势高位出现三波结构不追，一把回到解放前
- **弱水三千只取一瓢**（12-24）：只做能看懂的机会
- **空谷幽兰**（12-24）：稀缺完美交易机会
- **踏空比亏钱难受**：永远要有一定底仓
- **顺大势逆小势**：做上涨趋势的回调
- **不以账户盈亏做决策**（04-04）：基于盘面做决策，不是以盈亏额做决策。聪明资金先走再出利空
- **先见过钱**（04-04）：每个多头区间先把波段利润吃到，建立正向反馈。没见过钱的人很难存活
- **从小钱滚大靠波段，从大钱到自由靠长线**（04-04）：资金原始积累阶段做波段，体量上来后拿长线持有国运
- **散户做情绪，机构做逻辑**（04-20）：机构大钱基于宏观经济周期和产业逻辑做决策。散户永远在涨了高兴跌了骂
- **产业资本视角**（04-20/04-26）：把自己看作持有公司股权的股东，看到产业前景不是短期波动。持有1万股就是资本，老板在给你打工
- **投资者成长路径**（04-26）：短线（小资金熟悉市场）→波段（少妇战法B1-S1）→压行业做配置（大资金穿越周期）。不要拿后视镜做交易

### A3 仓位纪律
- 100万以下分4-5个仓位
- 每日压力测试：连续两个跌停不影响睡觉=最大仓位
- 用输得起的钱交易
- 7%以上才叫崩，1.6%叫波动
- **板块分散原则**（01-07）：与其给一个板块十次机会，不如给十个板块一次机会
- **主线/支线/答应框架**（04-12）：从资金数据判断板块轮动，主线（持续流入）→支线（潜力）→答应（调剂轮动）
- **全力追一个方向要小心见顶**（04-12）：当全市场都割掉白酒追科技时，科技就要到头了

### A4 每年抓5个完美B1（12-31）
250个交易日，用40个止损B1换5个大牛B1。执行这个策略就能稳定盈利。

### A5 常见错误
❌ 一跌就喊崩 ❌ 满仓单吊 ❌ 追高 ❌ 短线思维做中线 ❌ 对股票有感情 ❌ 把交易性机会当长线 ❌ 三天不涨不拍 ❌ 被套还补仓 ❌ 没有强宝宝乱切换 ❌ 研究一波流票 ❌ 舍不得放飞（02-19） ❌ 地产长期空头非要碰（03-11）

### A6 中阳线标准
券商4%-6%，白酒3.5%-6%，创业板20cm 7%以上，低波动票3%=中阳线。
