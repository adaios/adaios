## 总纲：少妇战法六步

```
①择时 → ②选股 → ③等B1 → ④设止损 → ⑤止盈放飞 → ⑥收队
```

对应执行口诀：
```
B1买 → 设止损 → 破了平 → 涨起来BBI线上两根中阳线放飞一半 → 看BBI线 → 跌破两根全离场
```

## 第一步：择时（什么时候做）

### 1.1 择时三要素（按顺序）
1. **基本面择时** → 判断大方向（宽松/紧缩，经济周期位置）
2. **资金面择时** → 看增量资金（活跃市值、Level2、国家队动向）
3. **技术面择时** → 找具体买点（K线形态、MACD、成交量）

### 1.2 A股底层认知（为什么A股这么"怂"）
1. **长期充分博弈**：各方参与者极致内卷
2. **交易机制不对等**：机构T+0 vs 散户T+1，各种门槛形成套利空间
3. **老麒麟会根深蒂固**：每个老票背后都有主，5-10年做一波
4. **洋务派主导大局**：留学背景基金经理看美股脸色
5. **政策驱动机制**：国家队和制度安排引导方向
6. **独特盈利模式**：不需要单边牛市，只需波动中获利

### 1.3 择时原则
- 宽松周期 + 经济拐点：**不能空仓**（至少半仓）
- 判断风险 > 收益：**空仓也是策略**
- A股盈利模式 = 波动，最适合低买高卖（少妇战法）
- 10年内不会再有924，不要抱不切实际幻想

### 1.4 策略回测原则
- 策略必须分**上涨/下跌/横盘**三个区间分开测试，不能混在一起
- 没有策略能在三种行情中都赚钱，找到策略最有效的阶段

## 第二步：选股（做什么票）

### 2.1 选股逻辑链
```
基本面看政策 → 政策看行业 → 行业看龙头 → 盘面语言确认
```

### 2.2 核心原则
- **不离开主线**：每个阶段市场有明确主线，偏离容易踏空或套牢
- **顺周期板块轮动顺序**：按经济周期正相关性排列，非银金融（券商）→ 白酒（消费）→ 其他顺周期
- **不碰下跌趋势板块**（如地产等大野图）

### 2.3 两条路线

**路线A：曼城阵容（大资金/中线配置）**
- 4231阵型：40%低波红利防守 + 30%前锋进攻 + 20%中场过渡 + 10%机动备用
- 每个票有明确角色（攻/守/对冲），知道它是干嘛的
- 必须配置不单调（1万也要分2-3个票）
- 2000万以上资金可适当集中

**路线B：少妇战法选股池（小资金/短线轮动）**
- 在主线板块内选股性好的票
- 不在下跌趋势票上做反弹（大爷图法则）
- 只需要一个池子，不追求自己挖掘

## 第三步：入场（怎么买）

### 3.1 B1 买入（核心入场方式）
```
条件：连续下跌 → N型转折 → KDJ大负值（J值达到大的负值，即"KDJ大幅值"） → 缩量确认
```
- B1不是抄底，是确认转折后才介入
- 配合MACD底背离 + BBI向上，确定性更强 → 做错也要做
- 一般第二天开盘买入（连续下跌后通常顺势低开）
- 特噗战法例外：有时需收盘买入（第二天容易高开）

- 比B1确定性更强，但止损幅度更大（7%-8%）
- 未熟练掌握B1止损前不建议做B2

### 3.4 SB1

### 3.5 特殊入场方式

**暴力特噗（突破入场）**

```
条件：放量大阳线突破前高 → 解放所有套牢盘
```
- 特噗后不能出现3/4阴量线

**深水炸弹（超跌反弹）**

- **特征**：股价没怎么跌，但KDJ快速下到大负值
- **判断**：这是洗盘，不是出货 → B1买入良机

## 第四步：设止损（怎么防）

### 4.1 只输一根K线（核心理念）
```
止损设在进场K线最低价下方几个价位（或1%），收盘跌破就走
```
- 永不套牢的基本保障
- Z家军永不被套

### 4.2 止损三级别（按碗大小）

| 级别 | 设置方法 | 适用 |
|:----|:---------|:-----|
| ① 敏感型 | 进场K线最低价下方3-5个价位 | 新手/小资金 |
| ② 趋势型 | 本阶段最低点或上个阶段最低点 | 有经验者 |
| ③ 盈亏比型 | 按盈亏比算（如向下10%向上50%=5:1） | 大心脏/大资金 |

### 4.3 中阳线标准（因票而异）

| 票类型 | 中阳线标准 |
|:-------|:----------|
| 低波动票（中石油） | 3%即为中阳线 |
| 券商 | 4%-6% |
| 台子（白酒） | 3.5% |
| 青花（白酒） | 4%-6% |
| 创业板（20cm） | 7%以上 |

### 4.4 其他止损方式
- **N型止损**：N型下跌前低下方
- **BBI拐头止损**：BBI转下 + 2日收不回
- **关键K止损**：关键K线最低价下方

## 第五步：止盈放飞（怎么减）

### 5.1 核心规则
```
BBI线上出现两根中阳线 → 减仓一半（卤煮/放飞）
```
- 减仓一半后成本大幅降低，剩余半仓拥有绝对成本优势
- 剩余半仓以BBI线为持股依据
- 卤煮=出货/清仓一部分，保证盈利落袋

### 5.2 涨势延续时的减仓节奏
- 第一波减半后，如果继续大涨可再减至1/4仓
- 三根以上连续大阳可考虑全部止盈

### 5.3 见顶信号也要减
- **顶部大风车**：连续大涨后放天量+长上影+长下影+螺旋桨（一根K线）→ 必须减仓
- **MACD顶背离**：价格创新高但红柱缩短/黄线走低 → 必须减仓
- **两个卖点**：S1（天量高位）+ S2（挑前高MACD顶背离）

## 第六步：收队（怎么走）

### 6.1 核心规则
```
跌破BBI线两根K线 → 全仓离场
```
- 一切以收盘价为准
- 不纠结、不幻想、不和股票谈恋爱

### 6.2 出场原则
- **顶部形态宁可信其有**
- **单顶不是顶**：大盘股一天出不完货，有二次冲高机会
- **下跌中的阳线叫反弹**：到压力位必须出，不是反转
- **行情末期不再做第二次**：完整波段走完后转震荡，不再参与

### 6.3 上涨趋势横盘后的最后一笔
- 每个上涨趋势变成横盘后，最后一次B1操作一定会止损
- 接受这是交易的一部分，遵守纪律比单次盈亏更重要

## 补充A：认知与纪律

### A1 四不原则（行为风控）
```
不追 → 不动 → 不慌 → 不乱摸
```

### A2 核心认知
- **机构思维**：严格止损、配置、长线格局、不被情绪左右
- **盈亏同源**：能单票赚大钱，就能单票亏大钱
- **积小胜为大胜**：不追求暴利，每次赚5%-10%，亏损控制在1%-5%
- **交易是持久战**：千锤百炼才能知行合一
- **顺大势逆小势**：做上涨趋势的回调，不做下跌趋势的反弹
- **踏空比亏钱更难受**：永远要有一定的底仓
- **永不套牢**：不被套就有持续参与市场的资格
- **永不炫技**：下跌趋势横盘（大爷图）不要碰

### A3 仓位纪律
- 100万以下分4-5个仓位
- 每日压力测试：连续两个跌停不影响睡觉 = 最大仓位
- 用输得起的钱交易
- 7%以上才叫崩，1.6%叫波动

### A4 常见错误
- ❌ 一跌就喊崩
- ❌ 满仓单调一只
- ❌ 期望买入后马上涨
- ❌ 追高（长期必亏行为，即使某次赚了也是错的）
- ❌ 短线思维做中线票
- ❌ 在大爷图上炫技
- ❌ 碰超小盘/ST
- ❌ 不离开主线
