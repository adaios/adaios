## 总纲：少妇战法六步

```
①择时 → ②选股 → ③等B1 → ④设止损 → ⑤止盈放飞 → ⑥收队
```

对应执行口诀：
```
B1买 → 设止损 → 破了平 → 涨起来BBI线上两根中阳线放飞一半 → 看BBI线 → 跌破两根全离场
```

## 第一步：择时（什么时候做）

### 1.1 择时三要素（按顺序）
1. **基本面择时** → 判断大方向（宽松/紧缩，经济周期位置）
2. **资金面择时** → 看增量资金（活跃市值、Level2、国家队动向）
3. **技术面择时** → 找具体买点（K线形态、MACD、成交量）

### 1.2 A股底层认知（为什么A股这么"怂"）
1. **长期充分博弈**：各方参与者极致内卷
2. **交易机制不对等**：机构T+0 vs 散户T+1，各种门槛形成套利空间
3. **老麒麟会根深蒂固**：每个老票背后都有主，5-10年做一波
4. **洋务派主导大局**：留学背景基金经理看美股脸色
5. **政策驱动机制**：国家队和制度安排引导方向
6. **独特盈利模式**：不需要单边牛市，只需波动中获利

### 1.3 择时原则
- 宽松周期 + 经济拐点：**不能空仓**（至少半仓）
- 判断风险 > 收益：**空仓也是策略**
- A股盈利模式 = 波动，最适合低买高卖（少妇战法）
- 10年内不会再有924，不要抱不切实际幻想

### 1.4 策略回测原则
- 策略必须分**上涨/下跌/横盘**三个区间分开测试，不能混在一起
- 没有策略能在三种行情中都赚钱，找到策略最有效的阶段

### 1.5 择时三要素（天时·地利·人和）
- **天时（宏观经济周期）**：当前中国处于扩张周期，全球经济下行
- **地利（政策周期）**：在中国最重要，有形的手比无形的手牛逼100倍。当前政策面高度呵护
- **人和（增量资金）**：最核心要素，用活跃市值监测

### 1.6 活跃市值（0AMV）判断体系
- **入场分级**：4%+阳线=主题级别，5%-8%=主线级别，7%-20%=十年一牛
- **离场法则**：单日跌幅≥2.3%绿柱=减仓离场
- **背离判断**：指数横盘但活跃市值下跌→存量资金出逃；指数未新高但活跃市值新高→还有第二波

### 1.7 十年一梦周期论
- 每十年一次资产重新定价：第一年转势→一年横盘（消灭分歧）→主升浪
- 当前处于类似2014年阶段——立柜改变趋势后横盘震荡

### 1.8 168万亿存款理论
- 银行体系168万亿居民存款，来30万亿就能到6000点
- 场外资金极度充裕，缺的是信心

## 第二步：选股（做什么票）

### 2.1 选股逻辑链
```
基本面看政策 → 政策看行业 → 行业看龙头 → 盘面语言确认
```

### 2.2 核心原则
- **不离开主线**：每个阶段市场有明确主线，偏离容易踏空或套牢
- **顺周期板块轮动顺序**：银行→券商→科技→保险→消费
- **不碰下跌趋势板块**（如地产等大野图）

### 2.3 两条路线

**路线A：曼城阵容（大资金/中线配置）**
- 4231阵型：40%低波红利防守 + 30%前锋进攻 + 20%中场过渡 + 10%机动备用
- 每个票有明确角色（攻/守/对冲），知道它是干嘛的
- 必须配置不单调（1万也要分2-3个票）
- 2000万以上资金可适当集中

**路线B：少妇战法选股池（小资金/短线轮动）**
- 在主线板块内选股性好的票
- 不在下跌趋势票上做反弹（大爷图法则）
- 只需要一个池子，不追求自己挖掘

### 2.4 六大赛道（2025下半年池子）
① 创新药（数字医疗） ② 新消费 ③ 固态电池 ④ AI+机器人（含AIDC/达链光模块/交换机） ⑤ 稳定币 ⑥ 可控核聚变

### 2.5 主题投资 vs 主线投资
- **主题**：持续时间短、资金容量小、波动大
- **主线**：持续时间长、资金容量大、走势稳健
- 主线容错率 > 主题容错率

### 2.6 机构资金定价权排序
国家队→社保→保险→公募→券商自营→私募（不重要）→麒麟会→游资→散户（无定价权）→外资（对大A不重要）

### 2.7 踏空鉴别框架
- **真踏空**：你懂的涨了你不在——心态问题
- **假踏空**：你不懂的涨了——正常

## 第三步：入场（怎么买）

### 3.1 B1 买入（核心入场方式）
```
条件：连续下跌 → N型转折 → KDJ大负值（J值达到大负值，即"KDJ大幅值"） → 缩量确认
```
- B1不是抄底，是确认转折后才介入
- 配合MACD底背离 + BBI向上，确定性更强 → 做错也要做
- 一般第二天开盘买入（连续下跌后通常顺势低开）
- KDJ.J < 13 可开始关注（根据各票股性调整）
- **买入三问**：①买入预期是什么（到前高？突破？填坑？）②盈亏比划不划算（短线至少3:1）③哪里设止损（前低/N型最低点/入场K线最低价下方）
- **呼吸三依据确认B1**：①向上N型结构 ②量能配合（放量吸/缩量呼）③KDJ辅助确认。三个缺一不可
- **短线三天不涨拍掉**：B1买入后三天没脱离成本区，拍掉换下一批
- **不要把交易性机会当长线**：破了止损位就走，不改成"不得不改的长线"

### 3.2 B2
- 比B1确定性更强，但止损幅度更大（7%-8%）
- 未熟练掌握B1止损前不建议做B2

### 3.3 B3
- 翻倍票第三个B1假破位往往是深蹲起跳的真机会

### 3.4 SB1（超级B1）
- 缩量横住→放量下跌→再缩量横住→勾大幅值=主升浪前最后一洗
- SB1是"忍出来的"，不是主动等的
- 图形最难看的时刻，大多数人在此被吓走
- 麒麟会洗盘博弈：主力借假破位把混进来的大资金筹码洗出来，快速拉回不给低位机会

### 3.5 特殊入场方式

**暴力特噗（突破入场）**
```
条件：放量大阳线突破前高 → 解放所有套牢盘
```
- 特噗后不能出现3/4阴量线
- 最好放量或平量延续

**深水炸弹（超跌反弹）**
```
条件：连续大幅下跌 → ADX拐头向上
执行：严格止损，3天不涨必走
```
- 仅日线级别，短线策略

### 3.6 洗盘识别
- **特征**：股价没怎么跌，但KDJ快速下到大负值
- **判断**：这是洗盘，不是出货 → B1买入良机

### 3.7 建仓波B1买入
- 底部横盘后连续中长阳（25%-50%），不放涨停
- 缩量回调B1可靠性高
- 建仓波不能有涨停（会引来游资抢筹打乱节奏）

### 3.8 拉升波B1买入
- 快速脱离成本区，涨幅50%-100%
- 急跌可赌B1，缓跌等第2/3个B1

### 3.9 冲刺波后处理
- 冲刺波顶部放量后B1一般不做
- 冲刺波后如果指数持续上涨，三波会循环接替，可等新一波建仓波
- 赌第三波必须设近止损（2%-3%），第三波容易转横盘

### 3.10 大盘不突破时填大坑回踩不碰
- 大盘没有突破的环境下，填大坑后回踩的票容易被封杀，不要参与

### 3.11 补票战法适用时机
- 补票战法（突破入场）最适合活跃市值红柱持续的市场环境
- 大盘突破环境下胜率高，大盘横盘环境下80%-90%是假突破

## 第四步：设止损（怎么防）

### 4.1 只输一根K线（核心理念）
```
止损设在进场K线最低价下方几个价位（或1%），收盘跌破就走
```
- 永不套牢的基本保障
- Z家军永不被套

### 4.2 止损三级别（按碗大小）

| 级别 | 设置方法 | 适用 |
|:----|:---------|:-----|
| ① 敏感型 | 进场K线最低价下方3-5个价位 | 新手/小资金 |
| ② 趋势型 | 本阶段最低点或上个阶段最低点 | 有经验者 |
| ③ 盈亏比型 | 按盈亏比算（如向下10%向上50%=5:1） | 大心脏/大资金 |

### 4.3 止损三原则（短线）
1. **赚钱的票不要做亏**：涨起来又跌回成本价，拍掉
2. **买入后该涨不涨拍掉**：成本区附近磨叽，拍掉（短线三天不涨拍掉）
3. **破掉止损价格拍掉**：收盘前确认，不盘中拍

### 4.4 正确交易 vs 错误交易
- 按照交易纪律做的单子，止损了也是**正确单子**
- 没按纪律做的单子，涨停了也是**错误单子**
- 交易不以盈亏论对错，以是否遵守纪律论对错

### 4.5 中阳线标准（因票而异）

| 票类型 | 中阳线标准 |
|:-------|:----------|
| 低波动票（中石油） | 3%即为中阳线 |
| 券商 | 4%-6% |
| 台子（白酒） | 3.5% |
| 青花（白酒） | 4%-6% |
| 创业板（20cm） | 7%以上 |

### 4.6 其他止损方式
- **N型止损**：N型下跌前低下方
- **BBI拐头止损**：BBI转下 + 2日收不回
- **关键K止损**：关键K线最低价下方

## 第五步：止盈放飞（怎么减）

### 5.1 核心规则
```
BBI线上出现两根中阳线 → 减仓一半（卤煮/放飞）
```
- 减仓一半后成本大幅降低，剩余半仓拥有绝对成本优势
- 剩余半仓以BBI线为持股依据
- 卤煮=出货/清仓一部分，保证盈利落袋

### 5.2 涨势延续时的减仓节奏
- 第一波减半后，如果继续大涨可再减至1/4仓
- 三根以上连续大阳可考虑全部止盈

### 5.3 见顶信号也要减
- **顶部大风车**：连续大涨后放天量+长上影+长下影+螺旋桨（一根K线）→ 必须减仓
- **MACD顶背离**：价格创新高但红柱缩短/黄线走低 → 必须减仓
- **两个卖点**：S1（天量高位）+ S2（挑前高MACD顶背离）

## 第六步：收队（怎么走）

### 6.1 核心规则
```
跌破BBI线两根K线 → 全仓离场
```
- 一切以收盘价为准
- 不纠结、不幻想、不和股票谈恋爱

### 6.2 出场原则
- **顶部形态宁可信其有**
- **单顶不是顶**：大盘股一天出不完货，有二次冲高机会
- **下跌中的阳线叫反弹**：到压力位必须出，不是反转
- **行情末期不再做第二次**：完整波段走完后转震荡，不再参与

### 6.3 正确交易 vs 错误交易（核心理念）
- 按照纪律执行的止损单 = 正确交易
- 没按纪律执行的涨停单 = 错误交易
- 不以单次盈亏评判交易，以是否遵守纪律评判

### 6.4 上涨趋势横盘后的最后一笔
- 每个上涨趋势变成横盘后，最后一次B1操作一定会止损
- 接受这是交易的一部分，遵守纪律比单次盈亏更重要

## 补充A：认知与纪律

### A1 四不原则（行为风控）
```
不追 → 不动 → 不慌 → 不乱摸
```

### A2 核心认知
- **机构思维**：严格止损、配置、长线格局、不被情绪左右
- **盈亏同源**：能单票赚大钱，就能单票亏大钱
- **积小胜为大胜**：不追求暴利，每次赚5%-10%，亏损控制在1%-5%
- **交易是持久战**：千锤百炼才能知行合一
- **顺大势逆小势**：做上涨趋势的回调，不做下跌趋势的反弹
- **踏空比亏钱更难受**：永远要有一定的底仓
- **永不套牢**：不被套就有持续参与市场的资格
- **永不炫技**：下跌趋势横盘（大爷图）不要碰
- **成功才是成功的妈妈**：看成功的案例建立信心，正向反馈
- **市场跟随原则**：跟随不是预判，市场没有证明你是对的=你是错的
- **"不要怕卖飞"**：连续快速上涨的票卖了就卖了，永远有下一批
- **"永远有18岁的"**：不恋战，市场永远有机会
- **"不要骂量化"**：量化是游资的对手盘，散户跟风骂量化没意义
- **"你唯一的负责人是你自己"**：做错后怪别人=这辈子赚不到钱

### A3 仓位纪律
- 100万以下分4-5个仓位
- 每日压力测试：连续两个跌停不影响睡觉 = 最大仓位
- 用输得起的钱交易
- 7%以上才叫崩，1.6%叫波动

### A4 常见错误
- ❌ 一跌就喊崩
- ❌ 满仓单调一只
- ❌ 期望买入后马上涨
- ❌ 追高（长期必亏行为，即使某次赚了也是错的）
- ❌ 短线思维做中线票
- ❌ 在大爷图上炫技
- ❌ 碰超小盘/ST
- ❌ 不离开主线
- ❌ 对股票有感情
- ❌ 把交易性机会当长线（破了止损位不走，改成"不得不改的长线"）
- ❌ 短线三天不涨不拍，死等
- ❌ 把单子盈亏当对错标准（赚钱=正确，亏钱=错误）

## 补充B：工具速查表

### B1 K线形态

| 形态 | 信号 |
|:----|:-----|
| 长下影阳线 | 多方完爆空方，继续看涨 |
| 长上影阳线 | 试探遇阻，守实体低点可看涨 |
| 长下影阴线 | 空头遇抵抗，不破实体高点继续看跌 |
| 长上影阴线 | 空头完爆多方，破低看空 |
| 十字星（趋势末端） | 反转概率高 |
| 标准K线（无影线） | 趋势延续 |

### B2 量价信号

| 信号 | 含义 |
|:----|:-----|
| 红肥绿瘦 | 阳量>阴量，健康持仓 |
| 3/4阴量线 | 突破后缩量75%，不好信号 |
| 缩半量 | 洗盘健康信号 |
| 放天量 | 可能见顶 |
| 缩量真实·放量可作假 | N型上涨找极致缩量，N型下跌找放量 |

### B3 图形分类

| 图形 | 判断 | 操作 |
|:----|:----|:-----|
| 大爷踏空图 | 上涨趋势横盘回调下沿 | 可以做B1 |
| 大爷图/大野图 | 下跌趋势横盘、重心向下 | 不能碰 |
| N型上涨 | 低点抬高+高点抬高 | 健康趋势 |
| N型下跌 | 高点降低+低点降低 | 等B1 |
| 顶部大风车 | 单日放天量+长上下影+螺旋桨 | 走错也要走 |
| 接力图形 | 底部连续放量阳线+顶部缩量 | 机构建仓完毕，回调B1可进 |

### B4 三波理论图

```
建仓波 →（缩量回调 B1）→ 拉升波 →（回调第2/3个B1）→ 冲刺波 →（顶部放量离场）
                                                              ↓ 如果指数持续上涨
                                                        回到建仓波→拉升波→冲刺波循环
```

## 补充C：每日复盘流程

### C1 复盘五步法
1. **看新闻** → 宏观政策、行业消息、国际形势
2. **看大盘** → 指数位置、活跃市值资金面
3. **看热点** → 今日领涨板块、涨停分布
4. **看持仓** → 已有票的BBI位置、止损是否触发
5. **选股+落实交易计划** → 在池子里选明天要做的票

### C2 B1每日选股流程
1. 在自选池/六大赛道内筛选
2. 找N型结构+缩量+勾大幅值
3. 计算买入三问（预期/盈亏比/止损）
4. 符合条件→做交易计划
5. 集合竞价后执行
