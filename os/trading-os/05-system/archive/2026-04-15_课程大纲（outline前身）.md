## 第一层：择时（什么时候做）— ⭐最重要

> 对应活跃市值（0AMV）判定体系

### 1.1 择时三要素
- 基本面择时 → 宏观周期判断
- 资金面择时 → 活跃市值、Level2、国家队
- 技术面择时 → K线形态、MACD、成交量

### 1.2 活跃市值（0AMV）
- 什么是活跃市值
- 4% 阳线 = 主题级别入场
- 5%-8% = 主线级别入场
- 7%-20% = 十年一牛
- **-2.3% 离场法则**
- 增量资金 vs 存量资金

### 1.3 择时原则
- 宽松周期+经济拐点：不能空仓
- 判断风险>收益：空仓也是策略
- 策略回测需分上涨/下跌/横盘三区间

## 第二层：选股（做什么票）

### 2.1 选股逻辑链
基本面看政策 → 政策看行业 → 行业看龙头 → 盘面语言确认

### 2.2 配置模式
- **曼城阵容（旧版）**：4231 固定比例配置
- **池子模式（新版）**：主线 → 支线 → 答应（灵活方向池）

### 2.3 核心原则
- 不离开主线
- 不碰下跌趋势板块（大爷图/大野图）
- 主题 vs 主线投资区分
- 坑不能做三条件
- 填坑出坑为核心战法
- 机构资金定价权排序

## 第三层：买入（怎么买）

### 3.1 B1 买点（核心入场方式）
- 条件：N型转折 + KDJ.W 大幅值（J<13）+ 缩量确认
- 买入三问：①预期？②盈亏比（≥3:1）③止损在哪？
- 短线三天不涨拍掉

### 3.2 B2 买点
- 确定性更强，止损幅度更大（7%-8%）

### 3.3 B3 买点
- 完整波段第三买点，只看白线（牵牛绳）

### 3.4 SB1 超级B1
- 缩量横住→放量下跌→再缩量横住→勾大幅值

### 3.5 特殊入场方式
- 暴力特噗（突破入场）
- 深水炸弹（超跌反弹，ADX拐头）
- 单针下20（公式指标）
- 补票战法

### 3.6 三波理论入场
- 建仓波B1（底部横盘后中长阳，不放涨停）
- 拉升波B1（快速脱离成本区）
- 冲刺波B1（谨慎，近止损）

### 3.7 砖形图（再确认工具）
- 砖形图六种定式
- 四块砖玩法
- **砖形图嫁接**：可验证B1/B2/B3的有效性
- **第四块砖放飞铁律**

### 3.8 买点分析依据（支持系统）
- 关键K线（暴力K / 倍量柱）
- ABC 结构
- N 型结构
- 对称结构
- 填坑出坑
- 长音短柱
- 接力图形
- 缩量真实·放量可作假
- 跟风盘成本交换理论

## 第四层：应对（买了以后怎么办）

### ⭐ 应对优先级（从最紧急到最终）

```
没涨 → 盈转亏 → 止损 → 放飞 → S1 → 滴滴 → 破白线 → 破黄线 → 死叉
```

### 4.1 没涨（最重要）
- 买入后没有快速脱离成本区
- 处理：直接拍掉
- 金句："市场没有证明你是对的时候，你就错了"

### 4.2 盈转亏
- 盘中浮盈变浮亏
- 处理：不等收盘，直接拍掉

### 4.3 止损体系
- 只输一根K线（核心理念）
- 止损三级别（敏感型/趋势型/盈亏比型）
- 止损三原则
- BBI拐头止损
- N型止损
- 关键K止损

### 4.4 放飞（止盈）
- BBI线上两根中阳线 → 减仓一半
- 见顶信号（顶部大风车 / MACD顶背离）
- 收队：跌破BBI线两根全离场

### 4.5 S1 五种出货方式
- 认识S1
- 如何识别

### 4.6 滴滴（阶梯量出货）
- 高位连续两根阴线下台阶
- 第二根必须走

### 4.7 破白线
- 牛股不破白线
- 破白线后到黄线找支撑

### 4.8 破黄线 / 击穿对手盘
- 黄线判断大哥在不在
- 击穿对手盘：忍一根

### 4.9 死叉
- 需要时间修复
