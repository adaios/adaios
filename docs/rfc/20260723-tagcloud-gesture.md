# UI 增强：标签图谱 + 双视图切换

> 日期：2026-07-23
> 涉及：前端 Flutter
> 状态：已实现

---

## 改动

### 1. 标签图谱视图（默认）

默认展示为力导向圆形图谱：
- 中心是频率最高的标签（大圆）
- 外围其他标签（大小依频率）
- 中心到外围有淡色连线
- 入场渐变动画（800ms）
- 点击任意标签 → 回到 Feed 并筛选

### 2. 列表视图切换

右上角 ☰ 列表 ↔ ✦ 图谱 切换按钮：
- 图谱视图（默认）：圆形气泡布局
- 列表视图：传统 Wrap 流式布局（原来的标签云）

### 3. 标签宇宙标题

- + 标签宇宙 + 总数
- 切换按钮在右上角

### 4. 全局空白拖拽（已放弃）

ListView 抢手势无法共存，放弃全局空白拖拽。
保持原有的三种切换方式：
- TopBar 下拉 → World B
- 输入栏上推 → World B
- Launcher 顶部把手下拉 → World A

### 未来方向

- 标签共现连线（需后端新增 `GET /api/v1/tags/graph`）
- 节点拖拽（Obsidian 风格弹性物理）
- 完整力导向布局引擎
