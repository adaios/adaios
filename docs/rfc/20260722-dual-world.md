# RFC: 双主页（Dual World）

> 日期：2026-07-22
> 涉及：前端 Flutter + 后端 Spring Boot
> 状态：草案

---

## 1. 当前问题

- App 只有一个页面（Feed），功能单薄
- 底部导航 4 个 Tab 空了 2 个（交易、记忆），装上后也显得空
- 用户想访问不同功能没有统一的入口

## 2. 方案：双主页

两个主页，通过不同手势切换。

### World A — Feed（日常阿呆）

不变。就是现在的样子：Feed 流 + 简报 + 输入栏。

```
┌──────────────────────────────────────┐
│  7/22·周二              👤          │
│                                      │
│  [简报卡片]                           │
│  [记录卡片]                           │
│  [对话卡片]                           │
│                                      │
│ ════════════════════════════════     │  ← 分隔线（始终在输入栏上方固定距离）
│  ┌──────────────────────────────┐   │
│  │ 📝 记录点什么...          ↑  │   │  ← 输入栏
│  └──────────────────────────────┘   │
└──────────────────────────────────────┘
```

**进入 World B 的方式：**
1. 从 TopBar 区域下拉（类似通知中心）
2. 从输入栏往上推（掀开地板）

### World B — Launcher（探索阿呆）

```
┌──────────────────────────────────────┐
│                                      │
│  ┌──────────────────────────────┐   │
│  │ 🔍 搜索记录、标签、记忆…  →  │   │  ← 和输入栏一致的外观风格
│  └──────────────────────────────┘   │
│                                      │
│  🧠  记忆              今日 +3      │
│     半导体·投资·天气                  │
│  ────────────────────────────────   │
│                                      │
│  🏷️  标签云            共 12 个    │
│     半导体 投资 天气 唐诗             │
│  ────────────────────────────────   │
│                                      │
│  👤  身份              阿呆          │
│     简洁直接 · 关注半导体             │
│  ────────────────────────────────   │
│                                      │
│  📅  时间线            2026-07      │
│     本月 18 条记录                    │
│  ────────────────────────────────   │
│                                      │
│  ──── 标签云 ────                    │
│                                      │
│  半导体  投资  天气  唐诗  测试       │
│  问候  关雎  诗经  后端  部署         │
│                                      │
└──────────────────────────────────────┘
```

**返回 World A 的方式：**
1. 从 TopBar 区域下拉（收回去）
2. 点搜索框左侧 ← 按钮

## 3. 页面切换：拖拽动画

```
              World A (y = 底部)
                   │
          ↑ 输入栏上推 / TopBar 下拉
                   │
              ┌────┴────┐
              │ 过渡态    │
              │ > 50% → B│
              │ < 50% → A│
              └────┬────┘
                   │
              World B (y = 顶部)
```

- Flutter 用 `AnimatedBuilder` + `GestureDetector`
- 拖拽超过屏幕 40-50% 吸过去，不到弹回
- 带弹性曲线（`Curves.easeOutCubic` 或类似 iOS 弹簧效果）
- 过渡时两个页面都在，一个淡出一个淡入

## 4. World B 各功能行右侧数据

| 行 | 右侧数据 | 数据来源 | API |
|:---|:---------|:---------|:----|
| 🧠 记忆 | 今日新增条数 · 标签预览 | Memory API | `GET /api/v1/memory?date=today` |
| 🏷️ 标签云 | 标签总数 · Top 3 标签 | TagIndex API | `GET /api/v1/tags`（新增）|
| 👤 身份 | 称呼 · 偏好摘要 | Identity API | `GET /api/v1/identity` ✅ |
| 📅 时间线 | 当前月份 · 本月记录数 | Timeline API | `GET /api/v1/timeline?limit=999` |

## 5. 后端改动

### 5.1 新增 API：`GET /api/v1/tags`

返回所有标签及其统计，供 Launcher 标签云 + 右侧预览使用。

**Response**

```json
{
  "tags": [
    {"name": "半导体", "count": 12, "lastAt": "2026-07-22T10:00:00"},
    {"name": "投资", "count": 8, "lastAt": "2026-07-21T14:30:00"},
    {"name": "天气", "count": 5, "lastAt": "2026-07-22T09:15:00"}
  ],
  "total": 12,
  "updatedAt": "2026-07-22T12:00:00"
}
```

**涉及文件**

| 文件 | 操作 | 说明 |
|:-----|:----:|:------|
| `TagIndexController.java` | **新建** | `GET /api/v1/tags` |
| `TagIndexService.java` | 改 | 暴露 `getAllTags()` 方法 |

### 5.2 搜索 API（本次不做，标记未来）

搜索后端 `kernel/search/` 目前仅有 `package-info.java`，需 Phase 2 实现。

### 5.3 无需改动

| API | 状态 | 用于 |
|:----|:----:|:-----|
| `GET /api/v1/identity` | ✅ | 身份行 |
| `GET /api/v1/memory?date=` | ✅ | 记忆行 |
| `GET /api/v1/timeline` | ✅ | 时间线行 |
