## 三、搜索 🔍

### 3.1 后端 API

需要新建 `GET /api/v1/search?q=半导体`

```json
{
  "results": [
    {
      "id": "rec_xxx",
      "type": "record",
      "title": "今天买了立昂微",
      "content": "今天买了立昂微，25块建仓...",
      "tags": ["投资", "半导体"],
      "dateTime": "2026-07-22T14:30:00",
      "highlights": ["今天买了**立昂微**，25块建仓..."]
    }
  ],
  "total": 1
}
```

### 3.2 后端改动

| 文件 | 操作 | 说明 |
|:-----|:----:|:------|
| `kernel/search/SearchService.java` | **新建** | 遍历 `data/records/` 全文匹配 |
| `kernel/search/SearchResult.java` | **新建** | 结果 DTO record |
| `interfaces/SearchController.java` | **新建** | `GET /api/v1/search?q=` |

**搜索范围**：`data/records/YYYY/MM/rec_xxx.md` 的文件内容

**实现方式**：
- 首次搜索时全量加载 records 文件（可缓存，10s 级别）
- 后续搜索在缓存中精确/模糊匹配
- 返回匹配的记录 ID、标题、内容片段、标签

### 3.3 UI 示意

```
┌──────────────────────────────────────┐
│  ← 🔍 搜索                           │
│                                      │
│  ┌──────────────────────────────┐   │
│  │ 半导体                    →  │   │  ← 搜索词
│  └──────────────────────────────┘   │
│                                      │
│  共 12 条结果                         │
│                                      │
│  ┌──────────────────────────────┐   │
│  │ 今天买了立昂微                │   │
│  │ 今天买了**立昂微**，25块建仓   │   │  ← 高亮匹配词
│  │ [投资] [半导体]   14:30      │   │
│  └──────────────────────────────┘   │
│                                      │
│  ┌──────────────────────────────┐   │
│  │ 半导体板块异动                │   │
│  │ **半导体**板块今天放量拉升    │   │
│  │ [投资] [半导体]   10:00      │   │
│  └──────────────────────────────┘   │
└──────────────────────────────────────┘
```

### 3.4 交互

- 进入页面自动聚焦键盘
- 输入后点 → 或回车触发搜索
- 无结果时显示"未找到相关记录"
- 结果列表可滚动，点击记录回 Feed 或记忆页
